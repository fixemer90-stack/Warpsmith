# Stimm Surge

«Стим-всплеск» — Круты вводят боевые стимуляторы, с удвоенной скоростью бросаясь на врага. Даёт +1 к броску заряда.

**Тактика:** критическая стратагема для агрессивной игры Kroot. У Koot нет врождённых бонусов к заряду — Stimm Surge заполняет этот пробел. Используйте на [[Kroot Carnivores]] для надёжного заряда с Deep Strike из 9" (нужно 8 на 2D6 → 7 с бонусом, ~58% успеха). [[Kroot Farstalkers]] тоже получают выгоду. Комбинируйте с Savage Maw для полного боевого потенциала: +1 к заряду + +1 S в ближнем бою. Отлично работает с [[Kroot Hounds]] — быстрые, дешёвые, с бонусом к заряду становятся отличными ти-ап-юнитами. Ссылки: [[Auxiliary Cadre]], [[T'au Empire]], [[Kroot Carnivores]], [[Kroot Farstalkers]], [[Kroot Hounds]], [[Savage Maw]].
