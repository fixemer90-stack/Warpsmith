# Cached Acquisition

Кэшированное приобретение! Даже уничтоженный отряд продолжает удерживать маркер цели — враг не может сразу перехватить контроль. В детачменте [[Explorator Maniple]] это критически важно для выполнения вторичных миссий, связанных с Acquisition.

**Тактика:** Используйте, когда отряд на ключевом маркере вот-вот будет уничтожен. Маркер остаётся вашим до тех пор, пока противник не начнёт/закончит ход, контролируя его. Это даёт вам время на контратаку. Комбинируйте с [[Reactive Safeguard]] для спасения отрядов с маркеров.
