# F4.7: Tooltips on Every Stat — M/T/SV/W/LD/OC

**Файлы:** `web/static/tooltips.js` (создать),
`web/templates/partials/tooltip_definitions.html` (создать),
`web/templates/base.html` (расширение — подключение tooltips.js)
**Зависимости:** UX.md (Tooltip design), F4.6 (Progressive Disclosure —
beginner mode показывает tooltips активнее)
**Часы:** 3h

**Цель (SMART):** Добавить информативные tooltips на каждую стату (Movement, Toughness,
Save, Wounds, Leadership, Objective Control) во всех местах отображения: карточки юнитов,
модалка, таблицы. Критерий: hover на стату показывает popup с названием, кратким
описанием и примером значений; tooltips отображаются через CSS transition (0.2s delay,
0.1s fade), не выходят за границы экрана (auto-flip), на мобильных — tap-to-show;
глоссарий доступен по клику на иконку `?`. 3 часа.

## Tooltip Manager — Alpine.js

```javascript
// web/static/tooltips.js
function tooltipManager() {
    return {
        visible: false,
        x: 0,
        y: 0,
        tooltip: null,
        statElements: [],
        activeStat: null,
        glossaryOpen: false,

        init() {
            this.registerStatElements();
        },

        registerStatElements() {
            // Все элементы с data-stat="M|T|SV|W|LD|OC" становятся триггерами
            document.querySelectorAll('[data-stat]').forEach(el => {
                el.classList.add('cursor-help');
                el.title = ''; // Remove native tooltip
            });
        },

        handleHover(event) {
            const target = event.target.closest('[data-stat]');
            if (!target) {
                this.hideTooltip();
                return;
            }

            const statKey = target.dataset.stat;
            const definition = STAT_TOOLTIPS[statKey];
            if (!definition) return;

            this.tooltip = definition;
            this.activeStat = statKey;

            // Position tooltip
            const rect = target.getBoundingClientRect();
            let tooltipX = rect.right + 8;
            let tooltipY = rect.top - 10;

            // Keep within viewport
            const tooltipWidth = 280; // max-w-xs
            const tooltipHeight = 300;
            if (tooltipX + tooltipWidth > window.innerWidth) {
                tooltipX = rect.left - tooltipWidth - 8;
            }
            if (tooltipY + tooltipHeight > window.innerHeight) {
                tooltipY = window.innerHeight - tooltipHeight - 8;
            }
            if (tooltipY < 0) tooltipY = 8;

            this.x = tooltipX;
            this.y = tooltipY;
            this.visible = true;
        },

        handleOut() {
            // Small delay to allow moving to tooltip
            setTimeout(() => {
                if (!this.activeStat) return;
                // Check if mouse is still over a stat element
                const hovered = document.querySelector('[data-stat]:hover');
                if (!hovered) {
                    this.hideTooltip();
                }
            }, 100);
        },

        handleClick(event) {
            const target = event.target.closest('[data-stat]');
            if (!target) return;

            const statKey = target.dataset.stat;
            if (this.activeStat === statKey && this.visible) {
                // Click = open glossary modal for this stat
                this.openGlossary(statKey);
            }
        },

        hideTooltip() {
            this.visible = false;
            this.tooltip = null;
            this.activeStat = null;
        },

        openGlossary(statKey) {
            const definition = STAT_TOOLTIPS[statKey];
            if (!definition) return;
            this.glossaryStat = definition;
            this.glossaryOpen = true;
        },
    };
}
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Tooltip выходит за правый край | Auto-flip влево от элемента |
| Tooltip выходит за нижний край | Поднимается вверх |
| Несколько статистик рядом | Каждая показывает свой tooltip |
| Мобильное устройство (touch) | tap = show tooltip, tap again = glossary |
| Быстрое движение мышью | 100ms debounce, без мерцания |
| Native title атрибут | Переопределён (title = '') |
