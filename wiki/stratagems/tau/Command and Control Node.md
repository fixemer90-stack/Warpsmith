# Command and Control Node

«Командно-контрольный узел» — баттлсьюты синхронизируют свои системы наведения через единую тактическую сеть, получая +1 BS к стрельбе.

**Тактика:** одна из сильнейших стратагем Retaliation Cadre. BS 4+ → BS 3+, BS 3+ → BS 2+. [[Crisis Battlesuits]] с BS 4+ становятся BS 3+, а [[Broadside Battlesuits]] (BS 4+) — BS 3+. [[Riptide Battlesuit]] с BS 4+ переходит на 3+. В комбинации с Deadly Precision (+1 S в 9") и Dropzone Clear (+1 to Hit после Deep Strike) можно достичь невероятной точности. Используйте на ключевом отряде баттлсьютов для уничтожения приоритетной цели. Отлично работает на [[Commander In Enforcer Battlesuit]] для максимальной эффективности. Ссылки: [[Retaliation Cadre]], [[T'au Empire]], [[Crisis Battlesuits]], [[Broadside Battlesuits]], [[Riptide Battlesuit]], [[Ghostkeel Battlesuit]].
