# F5.3: Rate limiting — slowapi
**Цель (SMART):** Подключить slowapi с лимитами: 30 req/min для анонимов, 100 req/min для авторизованных. Критерий: превышение лимита возвращает 429 JSON, заголовки Retry-After присутствуют, лимиты настраиваются через .env. 1 час.

**Файлы:**
- `main.py` (патч — middleware)
- `.env.example` (добавить RATE_LIMIT_*)
- `tests/test_rate_limit.py` (создать)

**Зависимости:** main.py, slowapi (pip)
**Часы:** 1h

---

## Установка

```bash
uv pip install slowapi
```

Добавить в `pyproject.toml`:

```toml
dependencies = [
    ...,
    "slowapi",
]
```

## Интеграция в main.py

```python
from slowapi import Limiter, _rate_limit_exceeded_handler
from slowapi.util import get_remote_address
from slowapi.middleware import SlowAPIMiddleware
from slowapi.errors import RateLimitExceeded
from backend.auth import get_current_user  # для user-based лимита

limiter = Limiter(
    key_func=get_remote_address,
    default_limits=[os.getenv("RATE_LIMIT_ANON", "30/minute")],
    storage_uri=os.getenv("RATE_LIMIT_STORAGE", "memory://"),
)


def create_app() -> FastAPI:
    app = FastAPI(...)

    # Rate limiting
    app.state.limiter = limiter
    app.add_exception_handler(RateLimitExceeded, _rate_limit_exceeded_handler)
    app.add_middleware(SlowAPIMiddleware)

    # ... остальная настройка ...
```

## Авторизованные пользователи — повышенный лимит

```python
from slowapi import Limiter
from slowapi.util import get_remote_address

def _rate_limit_key(request) -> str:
    """Ключ для rate limit: user_id если авторизован, иначе IP."""
    user = getattr(request.state, "user", None)
    if user:
        return f"user:{user.id}"
    return get_remote_address(request)


limiter = Limiter(
    key_func=_rate_limit_key,
    default_limits=["30/minute"],
)
```

На роутах:

```python
from slowapi import Limiter
from slowapi.util import get_remote_address

# Авторизованные получают 100 req/min
@router.get("/api/simulate")
@limiter.limit("100/minute")
async def simulate(request: Request):
    ...
```

## .env переменные

```env
# Rate limiting
RATE_LIMIT_ANON=30/minute
RATE_LIMIT_AUTH=100/minute
RATE_LIMIT_STORAGE=memory://
# Для production: RATE_LIMIT_STORAGE=redis://localhost:6379/0
```

## Ответ 429

```json
{
  "detail": "Rate limit exceeded: 30 per 1 minute",
  "retry_after": 42
}
```

Заголовки:
- `X-RateLimit-Limit: 30`
- `X-RateLimit-Remaining: 0`
- `X-RateLimit-Reset: 1714567890`
- `Retry-After: 42`

## Исключения

Эндпоинты, не подлежащие rate limiting:
- `/health` / `/` — проверка здоровья
- `/static/*` — статика (не роут FastAPI)

```python
@limiter.exempt
@router.get("/health")
async def health():
    return {"status": "ok"}
```
