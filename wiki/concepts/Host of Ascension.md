# Host of Ascension

Детачмент [[Genestealer Cults]] (культов генокрадов), посвящённый финальной стадии восстания — Вознесению. Ключевое правило: **A Plan Generations in the Making** — отряды GSC получают +1 к попаданию в первый раунд после прибытия из Reserves. Культ может использовать стратегемы для усиления атак из засад. Арми-рула: [[Cult Ambush]] (возврат сломленных отрядов).
