# Lethal Dosage

Летальная доза! Радиационные боеприпасы Skitarii получают Летальные Попадания — каждое попадание 6+ автоматически ранит. В детачменте [[Rad-Zone Corps]] это превращает любой отряд в серьёзную угрозу даже для тяжёлой техники.

**Тактика:** Отлична для [[Skitarii Vanguard]] с Radium Carbines — Lethal Hits в сочетании с их Anti-Infantry 4+ и радиационными эффектами. Также хороша для [[Onager Dunecrawler]] с Neutron Laser против тяжёлых целей — каждая 6+ на попадание автоматически ранит. Комбинируйте с [[Pre-Calibrated Purge Solution]] для переброса попаданий.
