# F4.2: Unit Modal — Squad Size, Loadout, Wargear Selection

**Файлы:** `web/templates/partials/unit_modal.html` (создать),
`web/static/unit_modal.js` (создать/расширить), `web/routes/api.py` (расширение)
**Зависимости:** F4.1 (Faction Browser), F1.1 (Unit Dataclass), F1.2 (Weapon Dataclass)
**Часы:** 6h

**Цель (SMART):** Реализовать модальное окно добавления юнита с full datasheet: выбор squad
size (min/max из wiki), выбор loadout (wargear options из YAML frontmatter), расчёт
итоговой PTS стоимости в реальном времени. Критерий: Alpine.js обновляет PTS при смене
loadout или размера, модалка показывает статы и оружие выбранного loadout, wargear
опции фильтруются по совместимости с юнитом. 6 часов.

## API Endpoint

```python
@router.get("/api/units/{unit_name}/detail")
async def unit_detail(unit_name: str):
    """
    Вернуть полные данные юнита для модалки.
    Включает: wargear_options, nob_options, squad_size, synergies,
    image, abilities, transport_capacity, leader_for, all weapons.
    """
    unit = registry.get_unit(unit_name)
    if not unit:
        raise HTTPException(404, detail="Unit not found")

    return {
        "name": unit.name,
        "faction": unit.faction,
        "category": unit.category,
        "points": unit.points,
        "movement": unit.movement,
        "toughness": unit.toughness,
        "save": unit.save,
        "wounds": unit.wounds,
        "leadership": unit.leadership,
        "oc": unit.oc,
        "abilities": getattr(unit, "abilities", []),
        "squad_size": getattr(unit, "squad_size", {"min": 1, "max": 1, "step": 1}),
        "wargear_options": getattr(unit, "wargear_options", []),
        "nob_options": getattr(unit, "nob_options", []),
        "weapons": [w.to_dict() for w in (getattr(unit, "weapons", []) or [])],
        "transport_capacity": getattr(unit, "transport_capacity", None),
        "leader_for": getattr(unit, "leader_for", []),
        "keywords": getattr(unit, "keywords", []),
        "faction_keywords": getattr(unit, "faction_keywords", []),
        "icon_url": icon_map.get_icon_url(unit.category),
        "color": icon_map.get_color(unit.category),
    }
```

## Alpine.js Logic (unit_modal.js)

```javascript
// web/static/unit_modal.js — расширение teamBuilder()
{
    // State
    showUnitModal: false,
    unitDetail: null,
    unitLoading: false,
    squadSize: 5,
    selectedLoadout: '',
    selectedNobOption: '',
    currentUnitName: '',

    openUnitModal(unitName) {
        this.currentUnitName = unitName;
        this.showUnitModal = true;
        this.unitLoading = true;
        this.unitDetail = null;

        fetch(`/api/units/${encodeURIComponent(unitName)}/detail`)
            .then(r => r.json())
            .then(data => {
                this.unitDetail = data;
                this.squadSize = data.squad_size.min || 5;
                this.selectedLoadout = data.wargear_options?.[0]?.name || '';
                this.selectedNobOption = '';
                this.unitLoading = false;
            })
            .catch(() => {
                this.unitLoading = false;
            });
    },

    get currentWeapons() {
        if (!this.unitDetail) return [];
        // Resolve weapons from selected loadout + nob option
        const loadout = this.unitDetail.wargear_options
            ?.find(o => o.name === this.selectedLoadout);
        const nobOpt = this.unitDetail.nob_options
            ?.find(o => o.name === this.selectedNobOption);

        let weaponNames = loadout?.weapons || [];
        // If nob option has weapons, they replace the default
        if (nobOpt?.replaces && nobOpt?.weapons) {
            weaponNames = weaponNames.filter(w => w !== nobOpt.replaces);
            weaponNames.push(...nobOpt.weapons);
        }

        return this.unitDetail.weapons
            ?.filter(w => weaponNames.includes(w.name)) || [];
    },

    get totalCost() {
        if (!this.unitDetail) return 0;
        const basePts = this.unitDetail.points;
        // Wargear option cost (per model)
        const loadout = this.unitDetail.wargear_options
            ?.find(o => o.name === this.selectedLoadout);
        const loadoutPts = loadout?.points || 0;
        // Nob option cost (one-time)
        const nobOpt = this.unitDetail.nob_options
            ?.find(o => o.name === this.selectedNobOption);
        const nobPts = nobOpt?.points || 0;

        return (basePts + loadoutPts) * this.squadSize + nobPts;
    },

    getQuickSizes() {
        if (!this.unitDetail?.squad_size) return [];
        const { min, max, step } = this.unitDetail.squad_size;
        const sizes = [];
        for (let s = min; s <= max; s += step) sizes.push(s);
        return sizes;
    },

    addUnitToRoster() {
        if (!this.unitDetail) return;
        this.roster.push({
            name: this.unitDetail.name,
            squad_size: this.squadSize,
            pts: this.totalCost,
            loadout: this.selectedLoadout,
            nob_option: this.selectedNobOption,
            weapons: this.currentWeapons,
            category: this.unitDetail.category,
        });
        this.showUnitModal = false;
        this.validationErrors = [];
    },
}
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Юнит без wargear options | Секция loadout скрыта, PTS = base × squad_size |
| Nob option с replaces, но оружия нет в текущем loadout | Опция скрыта или показана с предупреждением |
| Squad size min = max | Счётчик заблокирован, показано "Fixed size: X" |
| Ошибка загрузки /api/units/{name}/detail | В модалке красное сообщение, кнопка Retry |
| Модалка открыта, смена вкладки браузера | Сохраняется состояние (Alpine.js reactive) |
