# Grenade

Вместо обычной стрельбы отряд с ключевым словом GRENADE может метнуть гранату по врагу в пределах 6 дюймов. Один бросок D6 — на 4+ цель получает D6 смертельных ран (mortal wounds). **Не** считается стрельбой, поэтому не требует линии видимости и игнорирует укрытия.

**Ограничение:** можно использовать только один раз за фазу, и только если у отряда есть ключевое слово GRENADE. Не требует, чтобы отряд нёс гранату в снаряжении — ключевое слово заложено в профиле. Смертельные ранения не проходят спасброски, что делает эту стратагему отличным инструментом против высокобронных целей и персонажей.
