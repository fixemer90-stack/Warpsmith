# F3.10: Waaagh! — Orks Army Rule

**Файлы:** `backend/engine/scenario.py`, `backend/engine/ai/faction_ai.py`
**Зависимости:** F3.1 (Decision Engine), F2.5 (Game Loop), F3.2 (Faction AI)

## Цель (SMART)

Реализовать Waaagh! — фракционную способность Орков. Активируется один раз за бой в Command Phase, даёт всем ORKS: +1" Move, 5+ FNP, +1S в ближнем бою до конца раунда. Критерий: после активации Waaagh! damage output орков в melee увеличивается на ~17% (S+1), survivability на ~33% (5+ FNP). 3 часа.

## Механика

```
Command Phase → Ork player decides to call Waaagh!
  → Все ORKS юниты получают buffs до конца раунда:
    1. +1" Movement (M += 1)
    2. 5+ Feel No Pain
    3. +1 Strength в ближнем бою
  → Waaagh! used = True (нельзя повторно)
  → Лог: "WAAAGH! is called!"
```

## Implementation

### `backend/engine/scenario.py` — Command Phase

```python
def _command_phase(self):
    # ... существующая логика CP/Battle-shock ...

    # AI решение: когда активировать Waaagh!
    for player in self.state.players.values():
        if self._should_call_waaagh(player):
            self._activate_waaagh(player)

def _should_call_waaagh(self, player) -> bool:
    """AI решает, когда звать Waaagh!"""
    if self.state.waaagh_used:
        return False

    # Проверяем что это орки
    ork_units = [u for u in player.units.values()
                 if u.is_alive and u.faction == "orks"]
    if not ork_units:
        return False

    # Факторы решения:
    # 1. Много юнитов в charge range → звать сейчас
    # 2. Раунд 1-2: звать агрессивно (70% шанс)
    # 3. Раунд 3+: звать только если есть хорошие цели
    # 4. < 30% юнитов осталось → не звать (сохранить для next game)

    import random
    alive_ratio = len(ork_units) / max(1, len([u for u in player.units.values()]))
    charge_targets = sum(1 for u in ork_units
                         if any(_distance(u.position, e.position) <= 12
                               for e in self._get_enemy_units(player.player_id)))

    if self.state.current_round <= 2:
        return random.random() < 0.7
    elif charge_targets >= 2 and alive_ratio > 0.3:
        return random.random() < 0.4
    return False
```

### `backend/engine/scenario.py` — Apply Waaagh! buffs

```python
def _activate_waaagh(self, player):
    """Активировать Waaagh! для всех орков игрока."""
    self.state.waaagh_active = True
    self.state.waaagh_used = True
    self.state.waaagh_round = self.state.current_round

    for unit in player.units.values():
        if unit.faction != "orks" or not unit.is_alive:
            continue
        unit.waaagh_buffs = True
        self.state.game_log.append(f"WAAAGH! {unit.name} gains +1\"M, 5+FNP, +1S melee")

    self.state.game_log.append(f"*** WAAAGH! called by {player.player_id} in Round {self.state.current_round} ***")
```

### `backend/engine/combat.py` — Применить FNP и S+1

```python
def apply_waaagh_melee_bonus(strength: int, has_waaagh: bool) -> int:
    """Waaagh!: +1S в ближнем бою."""
    return strength + 1 if has_waaagh else strength

def apply_waaagh_fnp(fnp_value: int | None, has_waaagh: bool) -> int | None:
    """Waaagh!: 5+ FNP если нет лучшего."""
    if not has_waaagh:
        return fnp_value
    if fnp_value is None:
        return 5  # Waaagh! даёт 5+ FNP
    return min(fnp_value, 5)  # Используем лучшее
```

### `backend/engine/scenario.py` — Waaagh! expiry

```python
def _end_of_round(self):
    """В конце раунда сбрасываем Waaagh! buffs."""
    if self.state.waaagh_active and self.state.waaagh_round < self.state.current_round:
        self.state.waaagh_active = False
        for player in self.state.players.values():
            for unit in player.units.values():
                unit.waaagh_buffs = False
        self.state.game_log.append("Waaagh! ends")
```

## Интеграция в combat

```python
# В melee damage calculation:
effective_strength = apply_waaagh_melee_bonus(weapon.strength, attacker.waaagh_buffs)

# В damage application:
fnp = apply_waaagh_fnp(target.fnp, target.waaagh_buffs)
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Waaagh! уже использован | Не активируется повторно |
| Не-орки юниты (Gretchin?) | ВСЕ ORKS получают buffs (включая Gretchin) |
| Waaagh! + существующий FNP 4+ | Используется лучший (4+) |
| Юнит уничтожен до применения FNP | Не важно |
| Раунд закончился | Waaagh! buffs снимаются |
| AI не хочет звать Waaagh! | Ок, не заставляем |
