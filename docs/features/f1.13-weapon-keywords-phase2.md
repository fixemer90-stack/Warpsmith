# F1.13: Weapon Keywords — Phase 2

**Файл:** `backend/engine/combat.py`, `backend/engine/modifiers.py`
**Зависимости:** F1.6 (Combat Sequence), F1.7 (existing keywords: Sustained/Lethal/Devastating)

## Цель (SMART)

Реализовать 9 оставшихся weapon keywords из 10-й редакции, модифицирующих этапы Hit/Wound/Save/Damage. Критерий: каждый keyword имеет минимум 1 тест, проверяющий математически ожидаемый эффект. 6 часов.

## Keywords и этапы воздействия

| Keyword | Этап | Эффект | Приоритет |
|---------|------|--------|-----------|
| [BLAST] | Attacks | +1A за каждые 5 моделей в target (max +4). Против 10+ моделей — всегда max attacks | 1 |
| [HEAVY] | Hit | +1 to hit если модель не двигалась в этот ход | 2 |
| [TORRENT] | Hit | Автоматическое попадание (skip hit roll) | 1 |
| [PISTOL] | Eligibility | Можно стрелять в Engagement Range. Нельзя с другим оружием | 2 |
| [MELTA X] | Damage | +X damage на дистанции ≤ half range | 1 |
| [RAPID FIRE X] | Attacks | +X attacks на дистанции ≤ half range | 2 |
| [LANCE] | Wound | +1 to wound при заряде в этот ход | 2 |
| [PRECISION] | Allocation | Можно выбрать модель Character в target отряде | 3 |
| [ONE SHOT] | Usage | Оружие используется 1 раз за бой, затем помечается как used | 3 |

## Implementation

### `backend/engine/modifiers.py` — Blast modifier

```python
def apply_blast(attacks_dice: DiceExpr, target_models: int) -> DiceExpr:
    """Blast: +1 attack per 5 models in target unit (max +4)."""
    if target_models < 5:
        return attacks_dice
    bonus = min(4, target_models // 5)
    count, dice_type, modifier = attacks_dice
    return (count + bonus, dice_type, modifier)
```

### `backend/engine/modifiers.py` — Heavy modifier

```python
def apply_heavy(hit_prob: float, has_moved: bool) -> float:
    """Heavy: +1 to hit if stationary."""
    if has_moved:
        return hit_prob
    return min(5/6, hit_prob + 1/6)  # cap at 2+
```

### `backend/engine/modifiers.py` — Melta modifier

```python
def apply_melta(damage_dice: DiceExpr, distance: float, half_range: float, melta_bonus: int = 2) -> DiceExpr:
    """Melta X: +X damage at half range or less."""
    if distance > half_range:
        return damage_dice
    count, dice_type, modifier = damage_dice
    return (count, dice_type, modifier + melta_bonus)
```

### `backend/engine/modifiers.py` — Rapid Fire modifier

```python
def apply_rapid_fire(attacks_dice: DiceExpr, distance: float, half_range: float, rapid_fire_bonus: int) -> DiceExpr:
    """Rapid Fire X: +X attacks at half range."""
    if distance > half_range:
        return attacks_dice
    count, dice_type, modifier = attacks_dice
    return (count + rapid_fire_bonus, dice_type, modifier)
```

### `backend/engine/modifiers.py` — Lance modifier

```python
def apply_lance(wound_prob: float, has_charged: bool) -> float:
    """Lance: +1 to wound on charge."""
    if not has_charged:
        return wound_prob
    return min(5/6, wound_prob + 1/6)
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Blast против 1 модели | Бонус 0 |
| Blast против 20 моделей | Max +4 attacks |
| Heavy + moved | Без бонуса |
| Heavy + Advance | Moved = True → без бонуса |
| Torrent + Overwatch | Авто-попадание (6 не нужен) |
| Melta на нулевой дистанции (melee) | Не применяется (ranged only) |
| Lance + charge fail | has_charged = False → без бонуса |
| Pistol + другое оружие | Можно стрелять ТОЛЬКО Pistol (если выбрал Pistol, остальное нельзя) |
| One Shot + duplicate weapon name | Каждый экземпляр отслеживается отдельно |
| Precision + нет Character в отряде | Стандартное распределение |
| Несколько keywords на одном оружии | Применяются последовательно: Blast → Rapid Fire → Torrent → Heavy → Lance → Melta |
