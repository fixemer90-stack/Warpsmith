# F2.3: Line of Sight — Ray Casting

**Файл:** `backend/state/map.py` (расширение)
**Зависимости:** F2.2 (2D Map)
**Часы:** 4h

**Цель (SMART):** Реализовать LoS-проверку между двумя клетками через Bresenham ray casting с учётом блокирующего террейна. Критерий: open->open = True, open->blocked_by_wall = False, через light_cover = True (Obscuring rule), кэш на 1000+ запросов < 1ms. 4 часа.

## Core Algorithm

```python
from typing import Optional

def has_los(
    m: GameMap,
    x1: int, y1: int,
    x2: int, y2: int,
) -> bool:
    """Проверка LoS по Bresenham между (x1,y1) и (x2,y2)."""

    cache_key = (min(x1,x2), min(y1,y2), max(x1,x2), max(y1,y2))
    if cache_key in m.los_cache:
        return m.los_cache[cache_key]

    dx = abs(x2 - x1)
    dy = -abs(y2 - y1)
    sx = 1 if x1 < x2 else -1
    sy = 1 if y1 < y2 else -1
    err = dx + dy

    x, y = x1, y1
    blocked = False

    while True:
        if (x, y) != (x1, y1) and (x, y) != (x2, y2):
            feat = _get_blocking_feature(m, x, y)
            if feat is not None and feat.is_blocking_los:
                blocked = True
                break

        if (x, y) == (x2, y2):
            break

        e2 = 2 * err
        if e2 >= dy:
            err += dy
            x += sx
        if e2 <= dx:
            err += dx
            y += sy

    m.los_cache[cache_key] = not blocked
    return not blocked


def _get_blocking_feature(m: GameMap, x: int, y: int) -> Optional[TerrainFeature]:
    for f in m.features:
        if f.is_blocking_los and f.x <= x < f.x + f.width and f.y <= y < f.y + f.height:
            return f
    return None
```

## Distance Helpers

```python
def manhattan_distance(x1: int, y1: int, x2: int, y2: int) -> int:
    return abs(x1 - x2) + abs(y1 - y2)

def euclidean_distance(x1: int, y1: int, x2: int, y2: int) -> float:
    return ((x1 - x2) ** 2 + (y1 - y2) ** 2) ** 0.5

def inches_distance(x1: int, y1: int, x2: int, y2: int) -> float:
    """1 клетка = 1 дюйм."""
    return euclidean_distance(x1, y1, x2, y2)
```

## LoS Rules (10ed)

| Ситуация | LoS | Эффект |
|----------|-----|--------|
| Через open terrain | Да | — |
| Через light_cover (Obscuring) | Да | Цель получает cover |
| Через heavy_cover (Dense) | Да | -1 to hit |
| Через impassable / wall | Нет | — |
| Через занятую юнитами клетку | Да | Свои не блокируют |
| Из/в deploy zone | Да | — |
| На дистанции > weapon.range | Нет | Out of range |

## Тесты

```python
def test_los_open_terrain():
    m = create_default_map()
    assert has_los(m, 5, 30, 40, 30)

def test_los_blocked_by_wall():
    m = create_default_map()
    # Wall at (30, 15) size 2x10
    assert not has_los(m, 28, 5, 32, 25)

def test_los_through_cover():
    m = create_default_map()
    assert has_los(m, 10, 30, 25, 36)  # light_cover не блокирует

def test_los_cache():
    m = create_default_map()
    has_los(m, 0, 0, 47, 59)
    has_los(m, 0, 0, 47, 59)  # из кэша
    assert len(m.los_cache) == 1

def test_distance():
    assert abs(inches_distance(0, 0, 3, 4) - 5.0) < 0.01
    assert manhattan_distance(0, 0, 3, 4) == 7
```
