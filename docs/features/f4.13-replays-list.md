# F4.13: Replays List Page

**Файлы:** `web/templates/replays.html`, `web/routes/pages.py`
**Зависимости:** F3.6 (Replay Recording), F3.7 (Round Viewer), F3.8 (Result Screen)

**Цель:** Страница `/replays` со списком всех сохранённых реплеев, ссылками на Round Viewer и Result Screen.

## Реализация

- `GET /replays` → `replays.html` — Alpine.js загружает список через `GET /api/replays`
- Каждый реплей показывает: `game_id`, mission, deployment, seed
- Кнопки: «🎬 Replay» → `/replay/{game_id}`, «📊 Results» → `/result/{game_id}`
- Пустое состояние: "No replays yet" + ссылка на Scenario Setup
