## app.json (Dokku)

Не создан — Dokku настроен через [`deploy/dokku-setup.sh`](../../deploy/dokku-setup.sh). При необходимости:

```json
{
  "name": "warpsmith",
  "description": "WH40k Battle Simulator",
  "scripts": {
    "dokku": {
      "predeploy": "python -c 'from backend.db.database import db; db.migrate()'"
    }
  },
  "env": {
    "HOSTING": { "value": "true" },
    "DB_PATH": { "value": "/data/simulator.db" }
  }
}
```

## Dokku setup скрипт

**[`deploy/dokku-setup.sh`](../../deploy/dokku-setup.sh)** — полный скрипт в репозитории. Создаёт приложение, домен, letsencrypt, volume для SQLite, переменные окружения:

```bash
#!/usr/bin/env bash
# deploy/dokku-setup.sh — настроить Dokku-приложение
set -euo pipefail

APP="warpsmith"
DOMAIN="${1:-example.com}"

# Создать приложение
dokku apps:create "$APP" || true

# Установить домен и SSL
dokku domains:set "$APP" "$DOMAIN"
dokku letsencrypt:enable "$APP"

# Подключить volume для SQLite
dokku storage:ensure-directory "$APP-data"
dokku storage:mount "$APP" /var/lib/dokku/data/storage/"$APP-data":/data

# Переменные окружения
dokku config:set "$APP" HOSTING=true DB_PATH=/data/simulator.db

# Dockerfile деплой
dokku builder:set "$APP" selected dockerfile

echo "✓ Готово. git push dokku main для деплоя."
```

## Railway конфиг

**[`deploy/railway.json`](../../deploy/railway.json)** — конфиг Railway:

```json
{
  "build": {
    "builder": "DOCKERFILE",
    "dockerfilePath": "Dockerfile"
  },
  "deploy": {
    "startCommand": "uvicorn main:app --host 0.0.0.0 --port $PORT",
    "healthcheckPath": "/",
    "healthcheckTimeout": 100,
    "restartPolicyType": "ON_FAILURE"
  }
}
```

## systemd service (bare-metal)

**[`deploy/systemd.service`](../../deploy/systemd.service)** — для самостоятельного хостинга:

```ini
[Unit]
Description=Warpsmith Battle Simulator
After=network.target

[Service]
Type=simple
User=warpsmith
WorkingDirectory=/opt/warpsmith
Environment=HOSTING=true
Environment=DB_PATH=/var/lib/warpsmith/simulator.db
Environment=PORT=8000
ExecStart=/opt/warpsmith/.venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

## Переменные окружения (production)

| Переменная | Значение | Где задаётся |
|------------|----------|-------------|
| `HOSTING` | `true` | Всегда в production |
| `DB_PATH` | `/data/simulator.db` | Для volume-mounted SQLite |
| `ALLOWED_ORIGINS` | `https://ваш-домен.com` | Ограничить CORS |
| `JWT_SECRET` | `<random-64-char>` | Обязательно сменить |
| `PORT` | `8000` | Railway/докер задают авто |

## Безопасность

- JWT_SECRET генерируется через `openssl rand -hex 32`
- HTTPS настроен через letsencrypt (Dokku) или nginx (self-host)
- Production не использует reload
- SQLite volume монтируется отдельно, не в слое контейнера
