# F4.6: Progressive Disclosure — Beginner / Expert Modes

**Файлы:** `web/static/progressive_disclosure.js`,
`web/templates/base.html` (header toggle + CSS),
`web/templates/team_builder.html` (unit cards)
**Зависимости:** UX.md (Progressive Disclosure design)
**Часы:** 4h

**Цель (SMART):** Два режима отображения: Beginner (упрощённый — все 6 статов с подсказками,
Squad Size, без таблиц оружия и keywords) и Expert (полный datasheet с оружием, модификаторами
и тонкостями). Критерий: переключатель B/E в хедере, сохранение в localStorage, CSS-класс
на `<body>`, смена мгновенная без перезагрузки. 4 часа.

---

## Режимы

| Режим | Назначение | Что видно |
|-------|-----------|-----------|
| **Beginner** | Новый игрок, хочет быстро собрать армию | Все 6 статов (M,T,SV,W,LD,OC) с пояснениями, Squad Size, категория, PTS, beginner-advice. Без таблиц оружия, без keywords, без abilities. |
| **Expert** (default) | Опытный игрок | Полный datasheet: статы, таблица оружия, keywords, abilities, модификаторы, wargear. |

```html
    </button>

```

## Alpine.js Controller

```javascript
// web/static/progressive_disclosure.js
function progressiveDisclosure() {
    return {
        mode: 'expert',  // default

        loadMode() {
            const saved = localStorage.getItem('warpsmith_display_mode');
            if (saved && ['beginner', 'expert'].includes(saved)) {
                this.mode = saved;
            }
            this.applyMode();
        },

        setMode(newMode) {
            this.mode = newMode;
            try {
                localStorage.setItem('warpsmith_display_mode', newMode);
            } catch (e) { /* localStorage blocked */ }
            this.applyMode();
        },

        applyMode() {
            document.body.classList.remove('mode-beginner', 'mode-expert');
            document.body.classList.add('mode-' + this.mode);
            document.dispatchEvent(new CustomEvent('display-mode-changed', {
                detail: { mode: this.mode },
            }));
        },
    };
}
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Первый визит | Expert по умолчанию |
| Смена режима | CSS-класс на body → мгновенное скрытие/показ |
| localStorage заблокирован | Expert, без сохранения |
| Модалка открыта при смене | Alpine перерендерит x-show/x-if элементы |
