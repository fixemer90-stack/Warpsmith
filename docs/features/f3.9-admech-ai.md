**Файл:** `backend/engine/ai/mechanicus.py` (создать)

**Цель (SMART):** Реализовать фракционный AI профиль для Adeptus Mechanicus. Критерий: AdMech AI использует Canticles of the Omnissiah в ≥60% раундов (если доступны), держит Tech-Priests в радиусе 6" от Kastelan/ремонтируемых юнитов ≥70% ходов, выбирает цели с keyword Mönster/Vehicle приоритетнее Infantry (при равной дистанции). 4 часа.

## Фракционная механика

### Canticles of the Omnissiah

AdMech может активировать один Canticle за раунд. Влияние на decision weights:

| Canticle | Эффект | Влияние на AI |
|----------|--------|---------------|
| Shroudpsalm | Cover bonus | +survival_risk (держаться в террейне) |
| Litany of Electromancy | -1 AP в ближнем | +charge_score |
| Chant of the Remorseless | Ignore modifiers to hit | +shoot_aggro |
| Benediction of the Omnissiah | 5+ FNP | агрессивное позиционирование |
| Invocation of Machine Vengeance | +1 to hit против наиболее близкой цели | фокус на ближайшую |

### Doctrina Imperatives

- **Protector Imperative**: +1 BS, -1 M → AI предпочитает стрельбу, не двигается
- **Conqueror Imperative**: +1 WS, +1 advance/charge → AI агрессивен, заряжается

### Tech-Priest Synergy

Tech-Priests могут ремонтировать ближайшие AdMech Vehicle/Monster на D3 wounds.
AI должен:
- Держать Tech-Priest в радиусе 3" от ремонтируемого юнита
- Приоритет ремонта: юниты с <50% ран → Warlord → остальные
- Не удалять Tech-Priest от ключевых юнитов

## Overrides для Greedy Decision Engine

```python
ADMECH_WEIGHTS = {
    "kill_efficiency": 1.0,
    "threat_reduction": 0.9,       # выше — AdMech любит контроль
    "objective_value": 0.7,
    "survival_risk": 0.6,          # выше — ценят технику
    "synergy_bonus": 0.5,          # выше — Canticles + ремонт
}

# Целевые приоритеты
TARGET_PRIORITY = {
    "monster": 1.5,   # Anti-Monster/Vehicle оружие
    "vehicle": 1.3,
    "character": 1.0,
    "infantry": 0.8,
    "battleline": 0.6,
}
```

```python
def adjust_admech_weights(
    unit: UnitState, ctx: EvaluationContext
) -> dict[str, float]:
    """Скорректировать веса в зависимости от Canticle/Doctrina."""
    weights = dict(ADMECH_WEIGHTS)

    # Protector → больше стрельбы, меньше движения
    if ctx.state.current_phase == GamePhase.SHOOTING:
        weights["survival_risk"] *= 1.2  # осторожнее

    # Conqueror → больше зарядов
    if ctx.state.current_phase == GamePhase.CHARGE:
        weights["kill_efficiency"] *= 1.3

    return weights
```

## Edge Cases

| Edge Case | Ожидаемое поведение |
|-----------|---------------------|
| Tech-Priest один, все Vehicle мертвы | Идти в бой как Infantry Character |
| Canticle не выбран в этом раунде | Выбрать лучший для текущей ситуации |
| Kastelan в melee без поддержки Tech-Priest | Сделать Fall Back, если возможно |
| Два ремонтируемых юнита рядом | Приоритет — Warlord, затем самый дорогой |
| Protector + враг в charge range | Не заряжаться, стрелять с +1 BS |
| Все Canticles уже использованы | Действовать с базовыми весами |
| AdMech против AdMech | Игнорировать AdMech-specific weaknesses |
| Юнит с Invocation of Machine Vengeance | Атаковать ближайшую цель (forced target) |
