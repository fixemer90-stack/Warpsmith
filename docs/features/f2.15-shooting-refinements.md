# F2.15: Shooting Phase Refinements

**Файлы:** `backend/engine/scenario.py`, `backend/engine/ai/decision.py`
**Зависимости:** F2.5 (Game Loop), F1.13 (Pistol keyword), F2.13 (Cover)

## Цель (SMART)

Реализовать Big Guns Never Tire (MONSTER/VEHICLE стреляют в engagement), Look Out Sir (защита Characters), и Pistol в engagement range. Критерий: Vehicle в ближнем бою может стрелять с -1 to hit, Character за спинами союзников нельзя выбрать целью. 3 часа.

## Data Model

Всё на существующих моделях. Новых полей не требуется.

## Implementation

### 1. Big Guns Never Tire (BGNT)

```python
def _can_shoot_while_engaged(unit: UnitState, unit_model: Unit) -> bool:
    """MONSTER или VEHICLE могут стрелять даже в Engagement Range.

    - Могут стрелять в юнитов в своём Engagement Range
    - Могут стрелять в другие цели (с -1 to hit для HEAVY оружия)
    - Не могут стрелять BLAST оружием в engagement range
    """
    category = (unit_model.category or "").lower()
    return category in ("vehicle", "monster")


def _shooting_phase(state: GameState, unit_models: dict):
    for unit in active_player.units.values():
        if not unit.is_alive or unit.has_shot:
            continue

        if unit.is_engaged and not _can_shoot_while_engaged(unit, unit_models[unit.unit_id]):
            continue  # обычный юнит в ближнем бою — не стреляет

            if unit.is_engaged and "BLAST" in (weapon.keywords or []):
```

### 2. Look Out, Sir (LoSir)

```python
def _is_look_out_sir_protected(
    character: UnitState,
    friendly_units: list[UnitState],
    unit_models: dict,
) -> bool:
    """CHARACTER с ≤9 W защищён если есть союзный отряд в 3".

    Союзный отряд должен быть:
    - 3+ модели ИЛИ MONSTER/VEHICLE
    - Не в engagement range
    - В пределах 3" от character
    """
    char_model = unit_models.get(character.unit_id)
    if not char_model:
        return False

    # Только CHARACTER с ≤9 W
    cat = (char_model.category or "").lower()
    if cat not in ("character", "epic hero"):
        return False
    if char_model.wounds > 9:
        return False  # Ghazghkull и подобные

    # Ищем bodyguard
    for friend in friendly_units:
        if friend.unit_id == character.unit_id:
            continue
        if not friend.is_alive:
            continue
        if friend.is_engaged:
            continue

        friend_model = unit_models.get(friend.unit_id)
        if not friend_model:
            continue

        cat = (friend_model.category or "").lower()
        if cat in ("vehicle", "monster"):
            # MONSTER/VEHICLE = bodyguard при любом размере
            bodyguard = True
        elif friend.models_remaining >= 3:
            bodyguard = True
        else:
            continue

        if bodyguard:
            d = ((character.position[0] - friend.position[0]) ** 2 +
                 (character.position[1] - friend.position[1]) ** 2) ** 0.5
            if d <= 3:
                return True

    return False
```

### 3. Pistol в Engagement Range

```python
def _get_eligible_weapons(unit: UnitState, unit_model: Unit, is_engaged: bool) -> list[Weapon]:
    """Какое оружие можно использовать."""
    if not is_engaged:
        return unit_model.ranged_weapons  # всё доступно

    # В engagement range:
    # - BGNT: все кроме Blast
    # - Обычный юнит: только Pistol
    if _can_shoot_while_engaged(unit, unit_model):
        return [w for w in unit_model.ranged_weapons
                if "BLAST" not in (w.keywords or [])]

    pistols = [w for w in unit_model.ranged_weapons
               if "PISTOL" in (w.keywords or [])]
    if pistols:
        # Только PISTOL — другое оружие нельзя
        return pistols
    return []  # не может стрелять вообще
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Tactical Marine (INFANTRY) engaged | Не стреляет (нет Pistol) |
| Deff Dread (VEHICLE) engaged | Стреляет всем кроме Blast, -1 для Heavy |
| Warboss (CHARACTER, W=6) один | Можно выбрать целью (нет bodyguard) |
| Warboss + Boyz (20) в 2" | Look Out Sir — нельзя выбрать Warboss |
| Ghazghkull (CHARACTER, W=12) | Look Out Sir НЕ защищает (W>9) |
| Plasma Pistol + Bolter, engaged | Можно стрелять ТОЛЬКО Plasma Pistol |
| Vehicle engaged, Blast weapon | Blast нельзя, остальное можно |
