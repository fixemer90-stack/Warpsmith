# F3.1: Greedy Decision Engine — Target/Action Evaluation

**Файл:** `backend/engine/ai/decision.py` (создать)
**Зависимости:** F2.1 (Game State), F2.3 (LoS), F1.6 (Combat), F2.5 (Game Loop)
**Часы:** 6h

**Цель (SMART):** Реализовать жадный decision engine, который для заданного UnitState вычисляет наилучшее действие (цель, оружие, режим стрельбы/движения) на основе текущего GameState. Критерий: engine возвращает Action с максимальным score для любого допустимого состояния (минимум 6 тестов, покрывающих приоритизацию целей, выбор оружия, оценку движения и edge cases). 6 часов.

    mode: Optional[str] = None               # "normal", "overcharge", "grenade"

    """Контекст для оценки одного действия."""

## Decision Flow

```python
def choose_action(unit: UnitState, ctx: EvaluationContext,
                  weights: dict[str, float] = None) -> Action:
    """Главная точка входа: выбрать лучшее действие для юнита в фазе.

    Flow:
    1. Сгенерировать список кандидатов (все возможные действия)
    2. Оценить каждое через evaluate_action
    3. Выбрать с максимальным score
    4. Вернуть Action
    """
    candidates = _generate_candidates(unit, ctx)
    if not candidates:
        return Action(type=ActionType.HOLD, score=0.0,
                       rationale="no valid actions")

    weights = weights or DEFAULT_WEIGHTS
    for action in candidates:
        action = evaluate_action(unit, action, ctx)

    best = max(candidates, key=lambda a: a.score)
    return best


def _generate_candidates(unit: UnitState, ctx: EvaluationContext) -> list[Action]:
    """Сгенерировать все возможные действия в текущей фазе."""
    candidates = []

    if ctx.phase == "shooting":
        # Для каждого оружия — все цели в LoS + range
        for wi, weapon in enumerate(unit.ranged_weapons):
            if unit.is_engaged or unit.is_advancing:
                break  # нельзя стрелять
            for target in ctx.opponent.units:
                if target.squad_size <= 0:
                    continue
                if not _has_valid_los(unit, target, ctx.map_):
                    continue
                if not _in_weapon_range(unit, target, weapon):
                    continue
                candidates.append(Action(
                    type=ActionType.SHOOT,
                    target_id=target.unit_id,
                    weapon_index=wi,
                    mode="normal",
                ))

    elif ctx.phase == "charge":
        for target in ctx.opponent.units:
            if target.squad_size <= 0:
                continue
            dist = _distance(unit.position, target.position)
            if dist <= 12:  # max charge range
                candidates.append(Action(
                    type=ActionType.CHARGE,
                    target_id=target.unit_id,
                ))

    elif ctx.phase == "movement":
        # Двигаться к ближайшей точке / от угрозы
        for obj in ctx.state.mission.objectives:
            candidates.append(Action(
                type=ActionType.MOVE,
                target_position=(obj.x, obj.y),
            ))
        candidates.append(Action(
            type=ActionType.ADVANCE,
            target_position=(ctx.state.mission.objectives[0].x,
                             ctx.state.mission.objectives[0].y),
        ))

    return candidates
```

## Edge Cases

| Edge Case | Ожидаемое поведение |
|-----------|---------------------|
| Все цели уничтожены | Вернуть HOLD action |
| Юнит engaged, фаза shooting | Не генерировать SHOOT кандидатов |
| Юнит advancing без Assault | Не генерировать SHOOT (кроме Assault weapons) |
| Нет целей в LoS | HOLD или MOVE для улучшения позиции |
| Distance до цели > max charge range | Не генерировать CHARGE |
| Цель с FNP/Invul | Учесть в expected_damage |
| Overkill (expected >> remaining) | Штраф, выбрать другое оружие/цель |
| Оружие с Blast | Бонус против больших отрядов (squad_size >= 6) |
| Оружие с Torrent | Авто-hit, не зависит от BS модификаторов |
| Обнуление весов (все 0) | Все действия имеют score 0, выбрать первое |

def shooting_phase(state, map_, mission, log):
    for unit in state.active_roster().units:
        ctx = EvaluationContext(
            actor=unit, state=state,
            opponent=state.opponent_roster(),
            map_=map_, phase="shooting",
            turn=state.turn,

        action = choose_action(unit, ctx)
        if action.type == ActionType.SHOOT:
            damage = resolve_attack(unit, action.target_id,
                                    action.weapon_index, state)
            apply_damage(action.target_id, damage, state)
            log.events.append({
                "phase": "shooting", "actor": unit.unit_id,
                "action": action.type.value,
                "target": action.target_id, "damage": damage,
            })
