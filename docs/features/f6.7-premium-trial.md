# F6.7: 2-Week Premium Trial for All New Users

**Файлы:**
- `backend/auth/__init__.py` (User model + registration)
- `backend/db/database.py` (migration)
- `backend/billing/plans.py` (tier resolution)

**Зависимости:** Phase 6 (User tier system, Feature Gate)
**Часы:** 2h

**Цель (SMART):** Каждый новый пользователь при регистрации автоматически получает Premium на 14 дней. Через 2 недели tier возвращается на Free. Критерий: после регистрации `GET /api/me` возвращает `tier: "premium"` и `trial_ends_at`; через 14 дней tier становится `"free"`. 2 часа.

---

## Data Model

### DB — users table

Добавить колонку:

```sql
ALTER TABLE users ADD COLUMN trial_ends_at TIMESTAMP;
```

### User model

```python
class User:
    id: int
    email: str
    display_name: str
    tier: str                    # "free" | "premium"
    trial_ends_at: str | None    # ISO8601 timestamp или None
```

### Trial state

| Состояние | `tier` | `trial_ends_at` | Условие |
|-----------|--------|-----------------|---------|
| Активный триал | `premium` | дата в будущем | `now < trial_ends_at` |
| Триал истёк | `free` | дата в прошлом | `now >= trial_ends_at` |
| Нет триала | `free` | `NULL` | не было триала или куплен Premium |

## Implementation

### 1. DB Migration (`database.py`)

```python
# В migrate():
self.execute(
    "ALTER TABLE users ADD COLUMN trial_ends_at TIMESTAMP"
)
# Через contextlib.suppress(Exception) — если колонка уже есть
```

### 2. Registration (`auth/__init__.py`)

При регистрации нового пользователя:

```python
from datetime import datetime, timedelta, timezone

trial_ends = (datetime.now(timezone.utc) + timedelta(days=14)).isoformat()
tier = "premium"

db.execute(
    "INSERT INTO users (email, password_hash, display_name, tier, trial_ends_at) "
    "VALUES (?, ?, ?, ?, ?)",
    (email, pw_hash, display_name, tier, trial_ends),
)
```

### 3. Tier Resolution (`plans.py`)

`UserFeatures.for_user()` должно учитывать истечение триала:

```python
@classmethod
def for_user(cls, user) -> dict:
    if user is None:
        return cls.FREE.copy()

    tier = getattr(user, "tier", "free")

    # Проверить истечение триала
    trial_ends = getattr(user, "trial_ends_at", None)
    if tier == "premium" and trial_ends:
        from datetime import datetime, timezone
        now = datetime.now(timezone.utc)
        trial_end = datetime.fromisoformat(trial_ends)
        if now >= trial_end:
            tier = "free"

    return cls.for_tier(tier)
```

### 4. API Response (`auth/__init__.py`)

`User.to_dict()` должен включать `trial_ends_at`:

```python
def to_dict(self) -> dict:
    return {
        "id": self.id,
        "email": self.email,
        "display_name": self.display_name,
        "tier": self.tier,
        "trial_ends_at": self.trial_ends_at,
    }
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Триал истёк — пользователь логинится | `tier` автоматически возвращается `"free"` при проверке в `for_user()` |
| Пользователь купил Premium во время триала | `tier = "premium"`, `trial_ends_at = NULL` (триал не нужен) |
| Триал истёк, потом купил Premium | `tier = "premium"`, `trial_ends_at = NULL` |
| Повторная регистрация с тем же email | 409 Conflict — существующая логика |
