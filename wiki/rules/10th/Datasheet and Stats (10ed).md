# Datasheet and Stats (10ed)

Глоссарий характеристик юнитов в Warhammer 40,000 10-й редакции.

## Профиль отряда (Datasheet)

Каждый юнит имеет таблицу с шестью базовыми характеристиками:

| Стата | Название (RU) | Что значит |
|-------|---------------|------------|
| **M** | **Movement** (Движение) | Сколько дюймов может переместиться модель за фазу перемещения. Если 6" — может двигаться на 6 дюймов. Если стоит «—» — не может двигаться. |
| **T** | **Toughness** (Стойкость) | Насколько сложно ранить модель. Сравнивается с S (Strength) оружия при броске на ранение. Выше = живучее. |
| **SV** | **Save** (Защита) | Спас-бросок брони. Бросается D6 при получении ранения. SV 3+ = нужна 3+ на D6 чтобы спастись. Модифицируется AP оружия. |
| **W** | **Wounds** (Раны) | Сколько единиц урона модель может получить прежде чем будет уничтожена. W1 — гибнет от 1 раны, W12 (как у Ghazghkull) — нужно 12 ран. |
| **LD** | **Leadership** (Лидерство) | Для проверок Battle-shock. Бросается 2D6, результат ≤ LD = успех. LD 6+ = стандарт для пехоты, LD 5+ = элита, LD 4+ = герои. |
| **OC** | **Objective Control** (Контроль цели) | Сколько «весит» модель при контроле меток цели. Boyz OC2 = один бойз контролирует как два грота (OC1). Сумма OC всех моделей в пределах 3" от метки сравнивается с противником. |

## Профиль оружия (Weapon)

| Стата | Название | Что значит |
|-------|----------|------------|
| **Range** | Дальность | Максимальная дистанция стрельбы в дюймах. Melee = оружие ближнего боя. |
| **A** | **Attacks** (Атаки) | Сколько раз бросается кубик на попадание. A3 = три атаки. |
| **BS** | **Ballistic Skill** (Меткость) | Значение попадания для стрельбы. BS 4+ = нужно 4+ на D6. |
| **WS** | **Weapon Skill** (Ближний бой) | Значение попадания в рукопашной. WS 3+ = нужно 3+ на D6. |
| **S** | **Strength** (Сила) | Сравнивается с T цели при броске на ранение. Выше = легче ранить. |
| **AP** | **Armour Penetration** (Бронепробитие) | Модификатор спас-броска цели. AP -2 = защитник бросает SV+2 (SV 3+ → бросает 5+). |
| **D** | **Damage** (Урон) | Сколько ран снимает одна успешная атака. D3 = брось D6/2 (округление вверх). |

## Ключевые слова оружия

| Ключ | Эффект |
|------|--------|
| **Assault** | Можно стрелять после Advance (без штрафа) |
| **Heavy** | +1 к попаданию если модель не двигалась в этом ходу |
| **Pistol** | Можно стрелять даже в Engagement Range (ближний бой) |
| **Torrent** | Автоматическое попадание (не бросай на попадание) |
| **Twin-linked** | Переброс броска на ранение |
| **Blast** | +1 атака если отряд цели 5+ моделей; макс. атак если 10+ |
| **Rapid Fire X** | Удваивает число атак в половине дальности (или даёт +X) |
| **Melta X** | +X урона на половине дальности (или ближе) |
| **Lethal Hits** | 6 на попадание = автоматическое ранение (без броска на ранение) |
| **Sustained Hits X** | 6 на попадание = +X дополнительных попаданий |
| **Devastating Wounds** | 6 на ранение = смертельные раны (mortal wounds, игнорируют спас-бросок) |
| **Ignores Cover** | Цель не получает +1 к SV от укрытия |
| **Indirect Fire** | Можно стрелять без линии видимости, но -1 к попаданию |
| **Hazardous** | На 1 на попадание модель получает D3 mortal wounds |
| **One Shot** | Одноразовое оружие (один выстрел за бой) |
| **Precision** | Можно целиться в CHARACTER в отряде |

## Пример

**Ghazghkull Thraka:**
| M | T | SV | W | LD | OC |
|---|---|---|---|---|---|
| 5" | 7 | 2+ | 12 | 5+ | 5 |

Значит: ходит 5 дюймов, стойкость 7, спас-бросок 2+, 12 ран, лидерство 5+, контроль цели 5.

**The Great Claw:** A8 WS2+ S16 AP-4 D6 — 8 атак с WS2+, сила 16, бронепробитие -4, урон D6.
