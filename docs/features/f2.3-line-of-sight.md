# F2.3: Line of Sight — Ray Casting

**Файл:** `backend/state/map.py` (BattlefieldMap.has_los)
**Зависимости:** F2.2 (2D Map)
**Часы:** 4h
**Статус:** ✅ Done — Bresenham ray casting с кэшем

**Цель (SMART):** Реализовать LoS-проверку между двумя клетками через Bresenham ray casting с учётом блокирующего террейна. Критерий: open→open = True, через IMPASSABLE = False, кэш на уникальные пары, clear_los_cache() для сброса. 4 часа.

## Terrain Types и LoS

| TerrainType | Блокирует LoS | Эффект |
|-------------|---------------|--------|
| `OPEN_GROUND` | Нет | — |
| `DIFFICULT_TERRAIN` | Нет | Двойная стоимость движения |
| `DANGEROUS_TERRAIN` | Нет | Двойная стоимость + риск урона |
| `IMPASSABLE` | **Да** | Полная блокировка |

## Интеграция

LoS вызывается в `Scenario._shooting_phase()` при наличии `self.battlefield`:

```python
if self.battlefield is not None:
    if not self.battlefield.has_los(...):
        continue  # цель не видна, пропустить
```

## Тесты

```python
def test_los_open_terrain():
    """Open terrain с одной y — линия не пересекает IMPASSABLE."""
    m = BattlefieldMap(width=10, height=10,
                       terrain=np.full((10, 10), TerrainType.OPEN_GROUND, dtype=object))
    assert m.has_los(0, 6, 9, 6)

def test_los_blocked_by_wall():
    """IMPASSABLE колонна блокирует LoS."""
    m = BattlefieldMap(width=10, height=10,
                       terrain=np.full((10, 10), TerrainType.OPEN_GROUND, dtype=object))
    for y in range(5):
        m.set_terrain(5, y, TerrainType.IMPASSABLE)
    assert not m.has_los(2, 3, 8, 3)

def test_los_same_cell():
    """Старт = цель → всегда True."""
    m = BattlefieldMap(...)
    assert m.has_los(5, 5, 5, 5)

def test_los_target_on_wall():
    """Цель стоит на IMPASSABLE — всё равно видна."""
    m = BattlefieldMap(...)
    assert m.has_los(6, 3, 5, 3)  # цель на стене

def test_los_cache():
    """Повторный вызов возвращает кэшированный результат."""
    m = BattlefieldMap(...)
    m.has_los(0, 6, 9, 6)
    before = len(m._los_cache)
    m.has_los(0, 6, 9, 6)
    assert len(m._los_cache) == before
```
