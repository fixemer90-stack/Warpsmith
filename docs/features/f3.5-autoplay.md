| Компонент | Старый API (в spec / autoplay.py) | Актуальный (в коде) |
|-----------|----------------------------------|---------------------|
| `GameState` | `roster_a`, `roster_b`, `seed`, `mission`, `map_width`, `map_height` | `game_id`, `mission_name`, `players: dict[str, PlayerState]`, `terrain_map` |

## Edge Cases (целевые)

| Edge Case | Ожидаемое поведение |
|-----------|---------------------|
| Ростер невалиден (превышение PTS) | Вернуть InvalidRosterError |
| Оба AI одинаковые (Ork vs Ork) | Работает корректно |
| Юнит без оружия | Пропустить shooting phase |
| Все юниты одной стороны уничтожены | Завершить игру досрочно |
| Timeout превышен | Вернуть частичный результат с error |
| Seed = 0 (random) | Работает, но невоспроизводимо |
| Фракция без `ai:` секции в YAML | Fallback на `choose_action` (F3.1) |
| Ростер с 1 юнитом | Не падать |
