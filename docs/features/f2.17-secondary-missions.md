# F2.17: Secondary Missions — Fixed & Tactical

**Файлы:** `backend/state/secondary.py` (создать), `backend/engine/scenario.py` (интеграция)
**Зависимости:** F2.4 (Missions), F2.14 (Primary Missions)
**Часы:** 16h

**Цель (SMART):** Реализовать систему Secondary Missions из колоды Pariah Nexus —
9 Fixed (выбираются до игры) и 9 Tactical (вытягиваются случайно каждый раунд).
Каждая даёт VP за убийства, действия или контроль поля боя.
Критерий: 18 миссий в реестре, scoring через `score_secondary()`, integration в Command Phase, 3+ теста на категорию. 16 часов.

---

## Обзор

18 вторичных миссий из Pariah Nexus (2024). Делятся на:

| Категория | Кол-во | Когда выбирается |
|-----------|--------|-----------------|
| **Fixed** — Kill | 3 | До игры, действуют весь матч |
| **Fixed** — Action | 3 | До игры, действуют весь матч |
| **Fixed** — Board Control | 3 | До игры, действуют весь матч |
| **Tactical** — Board Control | 4 | Случайно, 1 карта в начале хода |
| **Tactical** — Kill | 3 | Случайно, 1 карта в начале хода |
| **Tactical** — Action | 2 | Случайно, 1 карта в начале хода |

## Data Model

### SecondaryMission

```python
@dataclass
class SecondaryMission:
    name: str                    # "Assassination", "Cleanse", etc.
    category: str                # "kill", "action", "board_control"
    mode: str                    # "fixed" | "tactical"
    description: str
    vp_reward: int = 5           # VP за выполнение
    max_vp: int = 20             # макс VP за игру с этой миссии
    conditions: dict[str, Any] = field(default_factory=dict)
    # conditions vary by category:
    #   kill: {"target": "character"|"monster_vehicle"|"horde"}
    #   action: {"location": "objectives"|"center"|"deployment_zone"|"edges"}
    #   board_control: {"zone": "enemy_deployment"|"quarters"|"no_mans_land"|"center"}
```

### PlayerSecondaryState

```python
@dataclass
class PlayerSecondaryState:
    fixed_mission: SecondaryMission | None = None   # выбрана до игры
    tactical_hand: list[SecondaryMission] = field(default_factory=list)  # текущая рука
    tactical_discard: list[SecondaryMission] = field(default_factory=list)
    completed_this_round: set[str] = field(default_factory=set)  # имена выполненных
    total_secondary_vp: int = 0
```

## Fixed Missions (9)

### Kill Secondaries

| Миссия | Цель | VP | Условие |
|--------|------|-----|--------|
| **Assassination** | Убить CHARACTER | 4 VP за unit | `target: "character"` |
| **Bring it Down** | Убить MONSTER/VEHICLE | VP = Wounds/2 (мин 2) | `target: "monster_vehicle"` |
| **Cull the Horde** | Убить INFANTRY 20+ моделей / 25+ W | 3 VP за unit | `target: "horde"` |

### Action Secondaries

| Миссия | Действие | VP | Условие |
|--------|----------|-----|--------|
| **Cleanse** | Action на controlled objective вне своей DZ | 4 VP | `location: "objectives"` |
| **Establish Locus** | Action в центре или DZ врага | 4 VP | `location: "center_or_enemy_dz"` |
| **Recover Assets** | Юниты в разных зонах (DZ, No Man's Land, вражеская DZ) | 5 VP за все 3 зоны | `location: "all_zones"` |

| Миссия | Зона | VP | Условие |

## Tactical Missions (9)

Вытягиваются случайно в начале Command Phase (1 карта за ход).
Игрок может держать до 3 карт в руке. Можно сбросить 1 карту в конце хода.

### Board Control

| Миссия | Условие | VP |
|--------|---------|-----|
| **Area Denial** | Юниты в 3" от центра, враг не в 6" | 5 VP |
| **Extend Battle Lines** | Контроль objective в своей DZ + нейтральной зоне | 5 VP |
| **Secure No Man's Land** | Контроль 2+ objectives в нейтральной зоне | 5 VP |
| **Defend Stronghold** | Контроль objective в своей DZ (не в 1 раунде) | 5 VP |

### Kill

| Миссия | Условие | VP |
|--------|---------|-----|
| **No Prisoners** | Убить любой вражеский unit | 3 VP за unit (макс 2/раунд) |
| **Overwhelming Force** | Убить unit в радиусе objective | 4 VP за unit |
| **Marked for Death** | Враг выбирает 3 своих unit — убей их | 5 VP за unit |

### Action

| Миссия | Условие | VP |
|--------|---------|-----|
| **Sabotage** | Action вне своей DZ (нейтральная = 3 VP, вражеская = 5 VP) | 3-5 VP |
| **Containment** | Action юнитами у разных краёв карты | 4 VP |

    # ... existing CP, battle-shock, engagement, primary scoring ...

## Scoring Logic

```python
def score_secondary(scenario: Scenario) -> dict[str, int]:
    vp = {}
    for pid, player in scenario.state.players.items():
        sec_state = scenario.secondary_states[pid]
        round_vp = 0

        # Fixed mission (scored every round)
        if sec_state.fixed_mission:
            round_vp += _score_fixed(scenario, pid, sec_state.fixed_mission)

        # Tactical hand (score from hand)
        for mission in sec_state.tactical_hand:
            if mission.name not in sec_state.completed_this_round:
                earned = _score_mission(scenario, pid, mission)
                if earned > 0:
                    sec_state.completed_this_round.add(mission.name)
                    round_vp += earned

        # Cap: max 40 VP from secondary missions total
        sec_state.total_secondary_vp = min(sec_state.total_secondary_vp + round_vp, 40)
        vp[pid] = round_vp

    return vp
```
