# Extinction Order

Приказ истребления! Tech-Priest активирует радиационное заражение маркера цели — морталки и Battle-shock врагам. В детачменте [[Rad-Zone Corps]] это отличный способ оказывать давление на ключевые точки поля боя.

**Тактика:** Используйте для ослабления вражеских отрядов, удерживающих маркеры цели. Технология работает на 24" — безопасная дистанция для [[Tech-Priest Manipulus]]. Каждый отряд на маркере получает отдельный бросок — против орд особенно эффективно. Комбинируйте с [[Lethal Dosage]] для финального удара.
