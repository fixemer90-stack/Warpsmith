# F4.10: Leaflet Battlefield Map v2 — Always-Visible Interactive Map

> **Superseded:** этот вариант карты оставлен в истории, но больше не используется в Scenario Setup. Активная карта — [F4.14 Strategic Battlefield Map](f4.14-strategic-battlefield-map.md): SVG, mission objectives, units из roster и реальный scale.


**Файлы:**
- `web/templates/partials/map_view.html` — Leaflet контейнер + легенда
- `web/static/map_view.js` — Alpine.js компонент карты
- `web/static/scenario_setup.js` — интеграция с выбором ростера/миссии
- `web/routes/api.py` — `/api/map/tiles` (террейн + objective positions)

**Зависимости:** F2.2 (Map), F2.4 (Missions), F4.5 (Canvas Map — заменён)

---

## Требования

1. **Карта всегда видна** — не скрывается за плейсхолдером. Показывается сразу при загрузке страницы с размером по умолчанию (Strike Force, 2000pts, 44×60).
2. **4 размера карт** — Combat Patrol (44×30), Incursion (44×44), Strike Force (44×60), Onslaught (44×90). Переключение через Game Format `<select>`.
3. **Террейн** — procedural generation: Open (тёмный), Light Cover (зелёный), Heavy Cover (тёмно-зелёный), Obstacle (чёрный), Difficult (коричневый), Deploy Zone (синий/красный полупрозрачный).
4. **Легенда** — компактная панель под картой: цветные квадратики + названия типов террейна.
5. **Objectives** — если у выбранной миссии есть objectives, они показываются жёлтыми кругами с тултипами.
6. **Юниты из ростеров** — при выборе ростеров юниты отображаются цветными точками в deploy zones (Player 1 — синие, Player 2 — красные).
7. **Deploy zones** — цветные полупрозрачные прямоугольники (Player 1 слева, Player 2 справа).

---

## UI Flow

```
1. Страница загружается
   → Карта ВИДНА: Leaflet 44×60, террейн, deploy zones
   → Легенда под картой
   → Ростеры: все доступны (без фильтра)

2. Пользователь выбирает Game Format
   → Карта перестраивается под новый размер
   → Objectives обновляются (если миссия выбрана)
   → Ростеры фильтруются по pts_limit ≤ формат

3. Пользователь выбирает миссию
   → Objectives появляются/исчезают на карте

4. Пользователь выбирает ростеры
   → Юниты отображаются точками в deploy zones
```

## map_view.js

```javascript
function mapView() {
    return {
        map: null,
        tileLayer: null,
        objectivesLayer: null,
        deployZoneLayer: null,
        unitLayer: null,
        gridSize: { width: 0, height: 0 },
        cellSize: 22,

        terrainColors: {
            0: '#2d2d44',  // Open
            1: '#4a7a3a',  // Light Cover
            2: '#2a5a2a',  // Heavy Cover
            3: '#1a1a2e',  // Obstacle
            4: '#5a4a2a',  // Difficult
            5: '#1e3a5f',  // Deploy Zone (P1 blue-ish)
            6: '#5f1e1e',  // Deploy Zone (P2 red-ish)
        },

        initMap(width, height, objectives, deployZones, units) {
            if (this.map) { this.map.remove(); this.map = null; }
            this.gridSize = { width, height };

            setTimeout(() => {
                const el = document.getElementById('battlefield-map');
                if (!el) return;

                const cell = this.cellSize;
                const mapH = height * cell;
                const mapW = width * cell;

                this.map = L.map('battlefield-map', {
                    crs: L.CRS.Simple,
                    minZoom: -2, maxZoom: 2,
                    zoomControl: true,
                    attributionControl: false,
                });

                this.map.setMaxBounds([[0, 0], [mapH, mapW]]);
                this.map.fitBounds([[0, 0], [mapH, mapW]]);

                // Координатная сетка (каждые 6 клеток)
                this._drawGrid(width, height, cell);

                fetch(`/api/map/tiles?width=${width}&height=${height}`)
                    .then(r => r.json())
                    .then(data => {
                        this._renderTiles(data, cell);
                        if (deployZones) this.showDeployZones(deployZones);
                        if (objectives) this.showObjectives(objectives);
                        if (units) this.showUnits(units);
                    });
            }, 100);
        },

        _drawGrid(width, height, cell) {
            const features = [];
            // Grid lines every 6 cells
            const step = 6;
            for (let x = 0; x <= width; x += step) {
                features.push({
                    type: 'Feature',
                    geometry: {
                        type: 'LineString',
                        coordinates: [[x * cell, 0], [x * cell, height * cell]],
                    },
                });
            }
            for (let y = 0; y <= height; y += step) {
                features.push({
                    type: 'Feature',
                    geometry: {
                        type: 'LineString',
                        coordinates: [[0, y * cell], [width * cell, y * cell]],
                    },
                });
            }
            L.geoJSON(
                { type: 'FeatureCollection', features },
                { style: { color: '#ffffff10', weight: 0.5 }, interactive: false }
            ).addTo(this.map);
        },

        _renderTiles(data, cellSize) {
            if (this.tileLayer) this.map.removeLayer(this.tileLayer);
            const tiles = data.tiles;
            const features = [];
            for (let y = 0; y < tiles.length; y++) {
                for (let x = 0; x < tiles[y].length; x++) {
                    const t = tiles[y][x];
                    const color = this.terrainColors[t] || '#333';
                    features.push({
                        type: 'Feature',
                        geometry: {
                            type: 'Polygon',
                            coordinates: [[
                                [x * cellSize, y * cellSize],
                                [(x + 1) * cellSize, y * cellSize],
                                [(x + 1) * cellSize, (y + 1) * cellSize],
                                [x * cellSize, (y + 1) * cellSize],
                            ]],
                        },
                        properties: { color, tileType: t },
                    });
                }
            }
            this.tileLayer = L.geoJSON(
                { type: 'FeatureCollection', features },
                {
                    style: (f) => ({
                        fillColor: f.properties.color,
                        color: '#ffffff08',
                        weight: 0.3,
                        fillOpacity: 0.85,
                    }),
                    onEachFeature: (f, layer) => {
                        const labels = ['Open', 'Light Cover', 'Heavy Cover', 'Obstacle', 'Difficult', 'Deploy Zone', 'Deploy Zone'];
                        layer.bindTooltip(labels[f.properties.tileType] || '', { sticky: true, opacity: 0.7 });
                    },
                }
            ).addTo(this.map);
        },

        showObjectives(objectives) {
            if (this.objectivesLayer) this.map.removeLayer(this.objectivesLayer);
            const c = this.cellSize;
            const markers = (objectives || []).map(obj =>
                L.circleMarker([obj.y * c + c/2, obj.x * c + c/2], {
                    radius: 10, color: '#fbbf24', fillColor: '#fbbf24', fillOpacity: 0.6,
                    weight: 2,
                }).bindTooltip(obj.label || 'Objective')
            );
            if (markers.length) this.objectivesLayer = L.layerGroup(markers).addTo(this.map);
        },

        showDeployZones(zones) {
            if (this.deployZoneLayer) this.map.removeLayer(this.deployZoneLayer);
            const c = this.cellSize;
            const rects = [];
            for (const [pid, cells] of Object.entries(zones || {})) {
                const color = pid === 'p1' ? '#3b82f6' : '#ef4444';
                for (const [x, y] of cells) {
                    rects.push(L.rectangle(
                        [[y * c, x * c], [(y + 1) * c, (x + 1) * c]],
                        { color, fillColor: color, fillOpacity: 0.15, weight: 0.5, interactive: false }
                    ));
                }
            }
            if (rects.length) this.deployZoneLayer = L.layerGroup(rects).addTo(this.map);
        },

        showUnits(units) {
            if (this.unitLayer) this.map.removeLayer(this.unitLayer);
            const c = this.cellSize;
            const markers = (units || []).map(u =>
                L.circleMarker([u.y * c + c/2, u.x * c + c/2], {
                    radius: 7, color: u.color || '#fff',
                    fillColor: u.color || '#fff', fillOpacity: 0.85,
                    weight: 2,
                }).bindTooltip(`${u.name}${u.faction ? ' (' + u.faction + ')' : ''}`)
            );
            if (markers.length) this.unitLayer = L.layerGroup(markers).addTo(this.map);
        },

        destroy() {
            if (this.map) { this.map.remove(); this.map = null; }
        }
    };
}
```

## scenario_setup.js (изменения)

```javascript
// В init():
init() {
    // Инит карты сразу (не ждать выбора формата)
    window.mapViewInstance = mapView();
    this.refreshMap();

    this.$watch('player1Roster', ...); // exist
    this.$watch('player2Roster', ...); // exist
    this.$watch('player2Generated', ...); // exist
    this.$watch('gameFormat', () => this.refreshMap());
    this.$watch('mission', () => this.refreshMap());
},

refreshMap() {
    const pts = parseInt(this.gameFormat) || 2000;
    const sizes = {500: [44,30], 1000: [44,44], 2000: [44,60], 3000: [44,90]};
    const [w, h] = sizes[pts] || [44, 60];

    // Get objectives for selected mission
    const objs = this.getMissionObjectives();

    // Get deploy zones from API
    window.mapViewInstance.destroy();
    window.mapViewInstance = mapView();
    window.mapViewInstance.initMap(w, h, objs, null, null);

    // После загрузки тайлов добавить юнитов
    setTimeout(() => this.updateMapUnits(), 300);
},

getMissionObjectives() {
    // Hardcoded objectives per mission (match backend create_mission logic)
    const w = 44, h = 60; // default size — will be scaled
    const pts = parseInt(this.gameFormat) || 2000;
    const sizes = {500: [44,30], 1000: [44,44], 2000: [44,60], 3000: [44,90]};
    const [mw, mh] = sizes[pts] || [44, 60];
    const cx = Math.floor(mw / 2), cy = Math.floor(mh / 2);

    if (this.mission === 'only-war') {
        return [
            {x: cx, y: cy, label: 'Center'},
            {x: Math.floor(cx/2), y: cy, label: 'Flank L'},
            {x: cx + Math.floor(cx/2), y: cy, label: 'Flank R'},
        ];
    }
    if (this.mission === 'take-and-hold' || this.mission === 'purge-the-foe') {
        return [
            {x: cx, y: cy, label: 'Center'},
            {x: Math.floor(cx/2), y: cy, label: 'Flank L'},
            {x: cx + Math.floor(cx/2), y: cy, label: 'Flank R'},
            {x: cx, y: Math.floor(cy/2), label: 'Home A'},
            {x: cx, y: cy + Math.floor(cy/2), label: 'Home B'},
        ];
    }
    return [];
},
```

```python
def test_map_tiles_all_sizes():
    """Все 4 размера должны отдавать корректные тайлы."""
    for w, h in [(44, 30), (44, 44), (44, 60), (44, 90)]:
        resp = client.get(f"/api/map/tiles?width={w}&height={h}")
        assert resp.status_code == 200
        data = resp.json()
        assert len(data["tiles"]) == h
        assert len(data["tiles"][0]) == w
        assert "deploy_zones" in data
        assert "p1" in data["deploy_zones"]
        assert "p2" in data["deploy_zones"]

def test_map_tiles_terrain_not_all_open():
    """Террейн не должен быть полностью пустым."""
    resp = client.get("/api/map/tiles?width=44&height=60")
    data = resp.json()
    all_open = all(
        cell == 0 for row in data["tiles"] for cell in row
    )
    assert not all_open, "Terrain should have non-open tiles"
```
