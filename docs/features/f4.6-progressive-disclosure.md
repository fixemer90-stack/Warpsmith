# F4.6: Progressive Disclosure — Beginner / Intermediate / Expert Modes

**Файлы:** `web/static/progressive_disclosure.js` (создать),
`web/templates/base.html` (расширение — header toggle),
`web/static/team_builder.js` (расширение)
**Зависимости:** UX.md (Progressive Disclosure design)
**Часы:** 4h

**Цель (SMART):** Реализовать три режима отображения данных: Beginner (только название,
PTS, категория, упрощённое описание), Intermediate (все статы, оружие, базовые keywords),
Expert (полный datasheet с модификаторами и тонкостями). Критерий: переключатель в хедере
сохраняет выбор в localStorage, Alpine.js применяет класс `mode-beginner/mode-intermediate/
mode-expert` к `<body>`, CSS/JS скрывают/показывают элементы в зависимости от режима,
смена режима не требует перезагрузки страницы. 4 часа.

## Alpine.js Controller

```javascript
// web/static/progressive_disclosure.js
function progressiveDisclosure() {
    return {
        mode: 'intermediate',  // default

        loadMode() {
            const saved = localStorage.getItem('display_mode');
            if (saved && ['beginner', 'intermediate', 'expert'].includes(saved)) {
                this.mode = saved;
            }
            this.applyMode();
        },

        setMode(newMode) {
            this.mode = newMode;
            localStorage.setItem('display_mode', newMode);
            this.applyMode();
        },

        applyMode() {
            // Remove all mode classes first
            document.body.classList.remove('mode-beginner', 'mode-intermediate', 'mode-expert');
            // Add current mode class
            document.body.classList.add(`mode-${this.mode}`);

            // Dispatch event for other controllers to react
            document.dispatchEvent(new CustomEvent('display-mode-changed', {
                detail: { mode: this.mode },
            }));
        },
    };
}
```

## Reactivity — Other Controllers

```javascript
// В team_builder.js — слушаем смену режима
document.addEventListener('display-mode-changed', (e) => {
    const mode = e.detail.mode;
    // Обновить отображение карточек
    if (mode === 'beginner') {
        // Показывать только название + PTS + категорию
        this.showSimplifiedCards = true;
    } else {
        this.showSimplifiedCards = false;
    }
});

// В unit_modal.js — перерисовка модалки при смене режима
document.addEventListener('display-mode-changed', () => {
    // Alpine.js автоматически перерендерит x-show/x-if при изменении
    // reactive переменной mode
});
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Первый визит | Intermediate по умолчанию |
| Смена с Expert на Beginner | Все expert-элементы скрываются мгновенно (CSS) |
| localStorage заблокирован | Режим Intermediate, без сохранения |
| Модалка открыта при смене режима | Alpine.js перерендеривает содержимое x-if |
| 3 переключателя одновременно | Только 1 активен (radio-style) |
