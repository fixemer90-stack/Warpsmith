# Command Phase

Первая фаза каждого хода.

## Порядок

1. **Получение CP** — активный игрок получает 1 командное очко (Command Point). CP тратятся на стратагемы.
2. **Battle-shock** — проверка для отрядов, потерявших более половины моделей (или более половины ран для одно-модельных отрядов) с начала текущего раунда:
   - Брось 2D6. Если результат ≤ LD отряда — проверка пройдена.
   - Если провалена — отряд Battle-shocked до начала вашей следующей командной фазы.
   - Battle-shocked: OC = 0, нельзя использовать стратагемы, влияющие на этот отряд.
   - Автоуспех: дубль 1 (без модификаторов) всегда успех.
3. **Начало фазы** — активация правил, срабатывающих «в начале командной фазы».
4. **Конец фазы** — активация правил, срабатывающих «в конце командной фазы».

## Отличия от 9ed

- В 9ed: отдельная Фаза морали. В 10ed: Battle-shock в командной фазе.
- В 9ed: CP давались за цели и лидеров. В 10ed: 1 CP автоматом каждый ход.
- В 9ed: лидерство (LD) было характеристикой 7-9. В 10ed: LD 4-7.
