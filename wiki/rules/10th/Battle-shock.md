# Battle-shock

Ключевая механика 10-й редакции, заменяющая мораль (Leadership/Morale) из 9-й. В начале каждой фазы командования (Command phase) игрок бросает 2D6 для каждого отряда, потерявшего хотя бы одну модель в этом раунде. Если результат меньше значения Лидерства (LD) отряда — тест пройден. Если равен или больше — отряд считается Battle-shocked до конца раунда.

## Эффекты Battle-shock

- **OC (Objective Control)** = 0 — отряд не может захватывать или удерживать метки целей
- Запрещено использовать стратагемы, влияющие на этот отряд (кроме [[Insane Bravery]])
- **Desperate Escape** — Fall Back с риском потери моделей (Battle-shocked отряды не могут Fall Back вообще)
