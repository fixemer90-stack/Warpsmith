# F2.8: Victory Points Tracking and End-Game Conditions

**Файл:** `backend/state/mission.py` (VPTracker, check_end_game) + `backend/state/game_state.py` (is_game_over)
**Зависимости:** F2.4 (Missions), F2.5 (Game Loop)
**Часы:** 2h

**Цель (SMART):** Подсчёт VP каждый раунд, проверка end-game условий, определение победителя. Критерий: после 5 раундов определяется победитель по VP; игра заканчивается досрочно при достижении 10 VP. Подсчёт поддерживает 3 режима (standard, progressive, kill_points) через `calculate_victory_points()`. 2 часа.

---

## Текущая реализация (MVP)

### VP добавляются каждый раунд

`_command_phase()` → `mission.calculate_victory_points()` → `player.victory_points += vp`

VP суммируются накопительно в `PlayerState.victory_points`.

### End-game

`GameState.is_game_over` (property):

```python
return self.current_round > self.max_rounds or any(
    player.victory_points >= 10 for player in self.players.values()
)
```

`GameState.winner` (property):

```python
max_vp = max(player.victory_points for player in self.players.values())
winners = [pid for pid, p in self.players.items() if p.victory_points == max_vp]
return winners[0] if len(winners) == 1 else None  # tie
```

### Условия окончания

| Условие | Реализация |
|---------|-----------|
| Max rounds | `current_round > max_rounds` ✅ |
| VP cap | ≥ 10 VP ✅ |
| Army wiped | ❌ не реализовано |

### apply_scoring()

```python
def apply_scoring(state, mission, vp_tracker) -> VPTracker:
    scorer = SCORING_MAP[mission.config.scoring_rule]
    round_vp = scorer(mission)
    for player_id, vp in round_vp.items():
        vp_tracker.add(player_id, vp)
    return vp_tracker
```
