# Tech-Priest

**Tech-Priest** — ключевое слово (Keyword) фракции [[Adeptus Mechanicus]]. Обозначает персонажей-техножрецов Культа Машины: от полевых инженеров до высоких лордов Марса.

## Что даёт ключевое слово

**Tech-Priest** keyword используется для:

1. **Стратагем и Энхансментов** — многие эффекты целятся только на TECH-PRIEST модели:
   - [[Extinction Order]] (Rad-Zone Corps) — цель: TECH-PRIEST
   - [[Omnissiah's Fury]] (Eradication Cohort) — SKITARII MARSHAL model only
   
2. **Лидерство** — Tech-Priests возглавляют отряды CULT MECHANICUS:
   - Dominus → [[Kataphron Breakers]], [[Kataphron Destroyers]]
   - Manipulus → [[Corpuscarii Electropriests]], [[Fulgurite Electropriests]], Kataphron
   - Enginseer → [[Servitors]]
   - Technoarcheologist → [[Skitarii Rangers]], [[Skitarii Vanguard]]

3. **Ауры** — каждый Tech-Priest даёт уникальную зону контроля:
   - Dominus: переброс ранений 1 для AdMech в 6"
   - Manipulus: +1 AP для BATTLELINE/CULT MECHANICUS в 6"
   - Enginseer: 5+++ против ран для VEHICLE в 6"

4. **Система Doctrina Imperatives** — все Tech-Priests имеют ключевое слово [[Doctrina Imperatives]], позволяющее переключать боевые протоколы.
