# F3.11: For The Greater Good & Markerlight — Tau Army Rule

**Файлы:** `backend/engine/scenario.py`, `backend/engine/ai/faction_ai.py`, `backend/engine/combat.py`
**Зависимости:** F3.1 (Decision Engine), F2.5 (Game Loop), F1.13 (Sustained Hits)

## Цель (SMART)

Реализовать For The Greater Good (FTGG) — систему Spotter/Guided для Тау. Один отряд становится Spotter (не стреляет), другой — Guided (+1 BS). Если Spotter имеет Markerlight, Guided также получает Sustained Hits 1. Критерий: пара Spotter-Guided увеличивает expected damage на ~33% (BS4+ → BS3+). 4 часа.

## Механика

```
Shooting Phase → Tau player:
  1. Выбрать одного вражеского юнита как цель FTGG
  2. Выбрать один отряд T'AU EMPIRE как Spotter (наводчик)
     - Spotter НЕ может стрелять в эту фазу
     - Spotter должен иметь LoS на цель
  3. Выбрать один отряд T'AU EMPIRE как Guided (наведённый)
     - Guided получает +1 BS против цели
     - Если Spotter имеет Markerlight → Guided также Sustained Hits 1
  4. Каждую фазу можно навести только ОДИН отряд
```

### `backend/engine/scenario.py` — Shooting Phase

```python
def _shooting_phase(self):
    for player in self.state.players.values():
        # ── FTGG: выбрать Spotter/Guided ──
        if self._is_tau_faction(player):
            spotter, guided, ftgg_target = self._select_ftgg_pair(player)
            if spotter and guided:
                spotter.is_spotter = True
                guided.is_guided = True
                guided.ftgg_target_id = ftgg_target.unit_id
                self.state.game_log.append(
                    f"FTGG: {spotter.name} spots for {guided.name} "
                    f"→ target {ftgg_target.name}"
                )

        # ── Стрельба ──
        for unit in player.units.values():
            if unit.is_spotter:
                self.state.game_log.append(f"{unit.name} is Spotter — cannot shoot")
                unit.has_shot = True
                continue

            # ... существующая логика выбора цели и стрельбы ...
            # При вычислении hit probability:
            if unit.is_guided and target.unit_id == unit.ftgg_target_id:
                hit_modifier += 1  # +1 BS от FTGG

        # ── Сброс FTGG состояния ──
        for unit in player.units.values():
            unit.is_spotter = False
            unit.is_guided = False
            unit.ftgg_target_id = None
```

### `backend/engine/ai/faction_ai.py` — FTGG AI

```python
def _select_ftgg_pair(player) -> tuple:
    """AI выбирает пару Spotter/Guided и цель FTGG.

    Стратегия:
    1. Выбрать самую опасную вражескую цель (high threat, low wounds)
    2. Guided = отряд с лучшим оружием против этой цели
    3. Spotter = ближайший отряд с Markerlight (если есть) или любой с LoS
    """
    enemies = _get_all_enemy_units(player)
    if not enemies:
        return None, None, None

    # Оцениваем ценность каждой вражеской цели
    best_target = max(enemies, key=lambda e: _threat_score(e))

    # Выбираем Guided: Tau отряд с максимальным expected damage против цели
    tau_units = [u for u in player.units.values()
                 if u.is_alive and not u.has_shot and u.faction == "tau"]
    if len(tau_units) < 2:
        return None, None, None  # нужна пара

    # Оцениваем expected damage каждого Tau отряда против цели
    # (используем существующий score_shoot из F3.1)
    guided = max(tau_units, key=lambda u: _ftgg_guided_score(u, best_target))

    # Spotter: предпочитаем с Markerlight
    spotters = [u for u in tau_units if u.unit_id != guided.unit_id]
    marker_spotters = [u for u in spotters if u.markerlight]

    if marker_spotters:
        spotter = min(marker_spotters, key=lambda u: _distance(u.position, best_target.position))
    elif spotters:
        spotter = min(spotters, key=lambda u: _distance(u.position, best_target.position))
    else:
        return None, None, None

    return spotter, guided, best_target
```

### `backend/engine/combat.py` — FTGG + Markerlight effects

```python
def apply_ftgg_bonus(hit_prob: float, is_guided: bool) -> float:
    """FTGG: +1 BS против цели."""
    if not is_guided:
        return hit_prob
    return min(5/6, hit_prob + 1/6)

def apply_markerlight(attacks_dice: tuple, is_guided: bool, spotter_has_markerlight: bool) -> tuple:
    """Markerlight: Sustained Hits 1 для Guided если Spotter имеет Markerlight."""
    if not is_guided or not spotter_has_markerlight:
        return attacks_dice
    # Sustained Hits 1: +1 авто-попадание на каждый critical hit (6)
    # Упрощённо: +1/6 от attacks к expected hits
    count, dice_type, modifier = attacks_dice
    bonus = count / 6  # expected sustained hits
    return (count, dice_type, modifier + bonus)
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Только 1 Tau отряд жив | FTGG недоступен (нужна пара) |
| Spotter с Markerlight | Guided получает +1 BS + Sustained Hits 1 |
| Spotter без Markerlight | Guided получает только +1 BS |
| Guided уже стрелял | Выбрать другого |
| Kroot отряд | Не считается T'AU EMPIRE (не участвует в FTGG) |
| Spotter в Engagement Range | Может быть Spotter если не стреляет |
| Цель FTGG вне LoS Spotter | Нельзя выбрать |
