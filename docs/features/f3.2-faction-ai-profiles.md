# F3.2: Faction AI Profiles — Wiki-driven Behaviour System

**Файлы:** `backend/engine/ai/faction_ai.py` (создать), `wiki/factions/*.md` (добавить ai: секцию)
**Зависимости:** F3.1 (Greedy Decision Engine)
**Часы:** 4h

**Цель (SMART):** Реализовать систему фракционных AI профилей, где все поведенческие параметры читаются из YAML `ai:` секции файлов `wiki/factions/<faction>.md`. Критерий: новый `.md` файл с `ai:` секцией = новая фракция с уникальным AI, ноль строк Python кода. Парсер читает 3 фракции (Orks, Tau, AdMech) и применяет их weights/behaviours/target_priority в choose_action(). 4 часа.

---

## YAML Schema

```yaml
# wiki/factions/<faction>.md — секция ai:
ai:
  weights:
    kill_efficiency: 1.0
    charge_aggression: 1.0
    survival_risk: 0.5
    objective_value: 0.6
    threat_reduction: 0.8
  behaviors:
    - id: faction_mechanic_name
      description: "Human-readable description"
      trigger:
        phase: command | shooting | charge | movement | fight | morale | any
        condition: "expression"  # evaluated against game state
        cooldown: int           # rounds between uses (0 = once per battle)
      effects:
        weights_override:       # temporary weight adjustments
          kill_efficiency: 1.5
        tags_buff: ["keyword"]  # e.g. "assault" on all weapons
        action_override:        # force a specific action type
          type: charge
          priority: 10
  target_priority:
    infantry: 1.0
    vehicle: 1.2
    monster: 1.3
    character: 1.0
    battleline: 0.8
    swarm: 0.5
  deployment:
    front_line: ["battleline", "infantry"]
    mid_field: ["elite", "character"]
    back_field: ["vehicle", "artillery"]
    reserve: ["fly", "deep_strike"]
```

## Data Model

```python
from dataclasses import dataclass, field
from typing import Optional
from enum import Enum


@dataclass
class BehaviorTrigger:
    phase: str
    condition: str = ""            # eval-выражение, опционально
    cooldown: int = 0              # 0 = одноразово
    round_min: int = 1
    round_max: int = 20


@dataclass
class BehaviorEffects:
    weights_override: dict[str, float] = field(default_factory=dict)
    tags_buff: list[str] = field(default_factory=list)
    action_override: Optional[str] = None


@dataclass
class Behavior:
    id: str
    description: str = ""
    trigger: BehaviorTrigger = field(default_factory=BehaviorTrigger)
    effects: BehaviorEffects = field(default_factory=BehaviorEffects)
    last_used_round: int = 0  # runtime state


@dataclass
class FactionAIProfile:
    faction_id: str
    weights: dict[str, float]
    behaviors: list[Behavior]
    target_priority: dict[str, float]
    deployment: dict[str, list[str]]
```

## Implementation

```python
# backend/engine/ai/faction_ai.py

def load_profile(faction_id: str, wiki_path: Path) -> Optional[FactionAIProfile]:
    """Загрузить AI профиль из wiki/factions/<faction>.md."""
    ...

def get_weights(profile: FactionAIProfile, ctx: EvaluationContext) -> dict[str, float]:
    """Вернуть актуальные weights с учётом активных behaviors."""
    weights = dict(profile.weights)
    for b in profile.behaviors:
        if _is_behavior_active(b, ctx):
            weights.update(b.effects.weights_override)
    return weights

def get_target_multiplier(
    profile: FactionAIProfile, target_keywords: set[str]
) -> float:
    """Вернуть множитель приоритета для цели по её keywords."""
    best = 1.0
    for kw in target_keywords:
        if kw in profile.target_priority:
            best = max(best, profile.target_priority[kw])
    return best

def _is_behavior_active(behavior: Behavior, ctx: EvaluationContext) -> bool:
    """Проверить, активен ли behavior в текущем контексте."""
    phase_ok = behavior.trigger.phase in ("any", ctx.phase.value)
    round_ok = behavior.trigger.round_min <= ctx.turn <= behavior.trigger.round_max
    cd_ok = (ctx.turn - behavior.last_used_round) > behavior.trigger.cooldown
    return phase_ok and round_ok and cd_ok
```

## Интеграция с F3.1

```python
from backend.engine.ai.faction_ai import load_profile, get_weights, get_target_multiplier

def choose_action_with_faction_ai(
    actor: UnitState, actor_unit: Unit, ctx: EvaluationContext
) -> Action:
    profile = load_profile(actor.faction, wiki.wiki_path)
    if profile:
        ctx.weights = get_weights(profile, ctx)
    return choose_action(actor, actor_unit, ctx)
```

## Edge Cases

| Edge Case | Ожидаемое поведение |
|-----------|---------------------|
| Файл фракции без `ai:` секции | Использовать DEFAULT_WEIGHTS из F3.1 |
| behavior с неизвестным phase | Игнорировать, логировать warning |
| Несколько behaviors активны одновременно | Суммировать effects.weights_override |
| target_priority без ключевого слова цели | Множитель 1.0 (нейтрально) |
| behavior condition с синтаксической ошибкой | Игнорировать behavior, логировать |
| Пустой deployment | Использовать Generic deployment |
| cooldown = 0 и behavior уже использован | Больше не активировать |
| Отрицательные веса | Не использовать behavior |
