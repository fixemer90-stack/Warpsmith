# F2.13: Cover & Terrain Effects

> **Follow-up requirement:** F2.13 is the simplified baseline for cover/terrain. The full Warhammer 40,000 10th Edition terrain model derived from `wiki/raw/papers/Terrain.md` is specified in [F2.18 — Terrain Mechanics 10e](f2.18-terrain-mechanics-10e.md).

**Файлы:** `backend/engine/combat.py`, `backend/state/map.py`
**Зависимости:** F2.2 (2D Map), F2.3 (LoS), F1.6 (Combat), F1.13 (Ignores Cover)

## Цель (SMART)

Реализовать Benefit of Cover (+1 SV) и связанные механики: Ignores Cover, Indirect Fire. Террейн на карте влияет на спасброски. Критерий: юнит за ruins получает +1 SV против ranged атак, если между ним и стрелком есть terrain. 4 часа.

## Implementation

### `backend/engine/combat.py` — Cover check

```python
def _has_cover(
    target_pos: tuple[int, int],
    shooter_pos: tuple[int, int],
    terrain_map: np.ndarray | None,
    target_category: str,
) -> bool:
    """Проверить, есть ли у цели Benefit of Cover.

    Cover даётся если:
    1. Цель INFANTRY — и любой terrain между shooter и target
    2. Цель полностью в terrain (любая категория — в woods)
    3. Цель в 1" от barricade
    """
    if terrain_map is None:
        return False

    tx, ty = target_pos
    sx, sy = shooter_pos

    # Проверка: цель стоит НА terrain
    tile = terrain_map[ty, tx]
    if tile == 2:  # woods — cover для всех
        return True

    # INFANTRY получает cover от ruins/woods/barricade на линии огня
    if target_category.lower() not in ("infantry", "character", "epic hero"):
        return False

    # Проверка: есть ли terrain между shooter и target (Bresenham)
    cells_between = _bresenham_line(sx, sy, tx, ty)
    for cx, cy in cells_between[1:-1]:  # исключаем start/end
        if terrain_map[cy, cx] in (1, 2, 4):  # ruins, woods, barricade
            return True

    return False
```

### `backend/engine/combat.py` — Apply Cover to save

```python
def compute_save(
    sv: int,
    ap: int,
    has_cover: bool,
    ignores_cover: bool,
) -> float:
    """Вероятность спасброска с учётом Cover и Ignores Cover.

    Cover: +1 SV (улучшает спасбросок на 1)
    Ignores Cover: отменяет Cover
    """
    if has_cover and not ignores_cover:
        sv = max(2, sv - 1)  # +1 save = -1 к所需ному броску (SV3+ → SV2+)

    effective_save = max(1, min(6, sv - ap))
    return (7 - effective_save) / 6
```

### `backend/engine/combat.py` — Indirect Fire

```python
def apply_indirect_fire(
    hit_prob: float,
    has_los: bool,
    target_has_cover: bool,
) -> float:
    """Indirect Fire: -1 to hit без LoS, дополнительный -1 если у цели Cover."""
    if has_los:
        return hit_prob  # прямой огонь — без штрафа

    # Без LoS: -1 to hit
    hit_prob = max(1/6, hit_prob - 1/6)

    # Если цель ещё и в Cover: дополнительный -1 (в пределах капа)
    if target_has_cover:
        hit_prob = max(1/6, hit_prob - 1/6)

    return hit_prob
```

### `backend/state/map.py` — Bresenham line

```python
def _bresenham_line(x0, y0, x1, y1) -> list[tuple[int, int]]:
    """Все клетки на прямой между двумя точками."""
    cells = []
    dx = abs(x1 - x0)
    dy = abs(y1 - y0)
    sx = 1 if x0 < x1 else -1
    sy = 1 if y0 < y1 else -1
    err = dx - dy

    while True:
        cells.append((x0, y0))
        if x0 == x1 and y0 == y1:
            break
        e2 = 2 * err
        if e2 > -dy:
            err -= dy
            x0 += sx
        if e2 < dx:
            err += dx
            y0 += sy
    return cells
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Террейн между shooter и target, target = INFANTRY | Cover (+1 SV) |
| Террейн между, target = VEHICLE | Нет Cover |
| Цель в woods | Cover для всех категорий |
| Shooter имеет Ignores Cover | Cover отменяется |
| Indirect Fire без LoS + цель в Cover | -2 to hit (кап ±1 всё равно, но суммируется с другими штрафами до капа) |
| Indirect Fire + Ignores Cover | Штраф только -1 (без дополнительного за Cover) |
| LoS заблокирован terrain | Indirect Fire всё равно может стрелять (со штрафом) |
| Нет terrain_map | Cover = False (без эффекта) |
| Melee атака | Cover не применяется |

## Тесты

```python
# tests/test_cover_terrain.py
import numpy as np

def test_infantry_behind_ruins_gets_cover():
    terrain = np.zeros((20, 30), dtype=int)
    terrain[5, 3] = 1  # ruins между
    assert _has_cover((5, 5), (5, 0), terrain, "Infantry") == True

def test_vehicle_behind_ruins_no_cover():
    terrain = np.zeros((20, 30), dtype=int)
    terrain[5, 3] = 1
    assert _has_cover((5, 5), (5, 0), terrain, "Vehicle") == False

def test_no_terrain_between_no_cover():
    terrain = np.zeros((20, 30), dtype=int)
    assert _has_cover((5, 5), (5, 0), terrain, "Infantry") == False

def test_ignores_cover_cancels_cover():
    assert compute_save(3, 0, True, True) == compute_save(3, 0, False, False)

def test_indirect_fire_no_los_minus1():
    assert apply_indirect_fire(4/6, False, False) == 3/6

def test_indirect_fire_no_los_cover_minus2():
    prob = apply_indirect_fire(4/6, False, True)
    assert prob == 2/6  # 4/6 - 2/6 кап не ниже 1/6

def test_cover_improves_save():
    # SV4+ → SV3+ с Cover
    assert compute_save(4, 0, True, False) == compute_save(3, 0, False, False)
```
