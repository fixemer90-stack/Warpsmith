## Data Model

```python
from dataclasses import dataclass, field
from typing import Optional

@dataclass
class Position:
    x: int
    y: int

@dataclass
class UnitState:
    """Состояние конкретного юнита / отряда на поле."""
    unit_id: str                    # уникальный ID (uuid)
    unit_name: str                  # ссылка на Unit.name
    faction: str
    squad_size: int                 # сколько моделей живо
    wounds_remaining: int           # суммарные раны отряда
    position: Position
    is_engaged: bool = False
    is_advancing: bool = False
    is_shooting: bool = False
    is_charging: bool = False
    is_fighting: bool = False
    is_battle_shocked: bool = False
    is_above_half_strength: bool = True
    cp_generated_this_turn: bool = False
    abilities_used: list[str] = field(default_factory=list)

@dataclass
class RosterState:
    """Одна армия."""
    player_id: int
    faction: str
    detachment: str
    units: list[UnitState]
    total_pts: int
    warlord_id: Optional[str] = None
    cp: int = 0
    command_priority: bool = False

@dataclass
class GameState:
    """Полное состояние игры."""
    roster_a: RosterState
    roster_b: RosterState
    round: int = 1
    phase: str = "command"
    sub_phase: str = ""
    turn: int = 1
    battle_round: int = 1
    victory_points: dict[int, int] = field(default_factory=lambda: {1: 0, 2: 0})
    mission: Optional[str] = None
    turn_history: list[dict] = field(default_factory=list)
    seed: int = 42

    def active_player(self) -> int:
        return 1 if self.turn == 1 else 2

    def active_roster(self) -> RosterState:
        return self.roster_a if self.turn == 1 else self.roster_b

    def opponent_roster(self) -> RosterState:
        return self.roster_b if self.turn == 1 else self.roster_a
```

## Phase Machine

```python
PHASE_ORDER = [
    "command",
    "movement",
    "shooting",
    "charge",
    "fight",
    "morale",
]

def next_phase(state: GameState) -> GameState:
    """Переключить фазу. В конце morale -> переключить игрока."""
    idx = PHASE_ORDER.index(state.phase)
    if idx < len(PHASE_ORDER) - 1:
        state.phase = PHASE_ORDER[idx + 1]
    else:
        if state.turn == 1:
            state.turn = 2
            state.phase = "command"
        else:
            state.turn = 1
            state.round += 1
            state.battle_round += 1
            state.phase = "command"
            _reset_activation_flags(state)
    return state
```

def game_state_to_json(state: GameState) -> str:
    return json.dumps(asdict(state), indent=2, default=str)

## Фазы

### Command Phase
- Оба игрока получают CP (1 в начале раунда, +1 за warlord)
- Battle-shock тесты для отрядов ниже половины

### Movement Phase
- Normal Move: до M дюймов
- Advance: Move + D6, нельзя стрелять (кроме Assault)
- Fall Back: отход из engagement range

### Shooting Phase
- Выбрать цель в LoS и в range
- Hit -> Wound -> Save -> Damage

### Charge Phase
- Ролл 2D6, нужно дотянуться до цели
- Engagement range: 1"

### Fight Phase
- Alternating activations
- Melee attacks

### Morale Phase
- Battle-shock для отрядов, потерявших модель
