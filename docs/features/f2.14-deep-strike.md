# F2.14: Deep Strike & Strategic Reserves

**Файлы:** `backend/engine/scenario.py`, `backend/engine/ai/deployment.py`
**Зависимости:** F2.2 (2D Map), F2.1 (Game State), F3.4 (Deployment AI)

## Цель (SMART)

Реализовать правило Deep Strike: юниты с этим правилом могут начать игру вне поля и прибыть в фазе движения (Round 2+), размещаясь >9" от врагов. Критерий: юнит с Deep Strike размещается на карте во 2-м раунде в своей deployment zone, не ближе 9" к врагу. 3 часа.

## Implementation

### `backend/engine/scenario.py` — Deployment

```python
def deploy_game(self):
    """Развёртывание: обычные юниты на поле, Deep Strike → в резерв."""
    for player in self.state.players.values():
        for unit in player.units.values():
            if unit.deep_strike:
                unit.in_reserves = True
                unit.position = (-1, -1)  # off-map
                self.state.game_log.append(f"{unit.name} starts in Deep Strike reserve")
```

### `backend/engine/scenario.py` — Arrival in Movement Phase

```python
def _movement_phase(self):
    # ... существующая логика движения ...
    
    # Deep Strike arrival (Round 2+)
    if self.state.current_round >= 2:
        for player in self.state.players.values():
            reserves = [u for u in player.units.values() if u.in_reserves]
            for unit in reserves:
                pos = self._find_deep_strike_position(unit, player.player_id)
                if pos:
                    unit.position = pos
                    unit.in_reserves = False
                    unit.arrived_this_turn = True
                    unit.has_moved = True  # не может двигаться после прибытия
                    self.state.game_log.append(
                        f"{unit.name} arrives from Deep Strike at ({pos[0]}, {pos[1]})"
                    )

def _find_deep_strike_position(self, unit, own_player_id) -> tuple | None:
    """Найти валидную позицию для Deep Strike: >9" от врагов, в deployment zone."""
    import random
    
    # Определяем deployment zone
    is_player1 = list(self.state.players.keys()).index(own_player_id) == 0
    if is_player1:
        zone_x = range(0, self.state.map_width // 3)
    else:
        zone_x = range(2 * self.state.map_width // 3, self.state.map_width)
    
    # Собираем позиции всех врагов
    enemy_positions = []
    for pid, player in self.state.players.items():
        if pid == own_player_id:
            continue
        for eu in player.units.values():
            if eu.is_alive and eu.position != (-1, -1):
                enemy_positions.append(eu.position)
    
    # Пробуем до 50 случайных позиций
    for _ in range(50):
        x = random.choice(list(zone_x))
        y = random.randint(0, self.state.map_height - 1)
        
        # Проверка: не ближе 9" к врагам
        too_close = False
        for ex, ey in enemy_positions:
            if ((x - ex) ** 2 + (y - ey) ** 2) < 81:  # 9^2 = 81
                too_close = True
                break
        
        if not too_close:
            # Проверка: клетка не занята terrain (непроходимый)
            if self.state.terrain_map is not None:
                if self.state.terrain_map[y, x] >= 5:  # assuming 5+ = impassable
                    continue
            return (x, y)
    
    return None  # не нашли позицию — остаётся в резерве
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Round 1 | Deep Strike юниты не могут прибыть |
| Нет валидной позиции (>9" от всех врагов) | Остаётся в резерве, попробует в следующем ходу |
| Все враги уничтожены | Можно разместиться где угодно в своей zone |
| Карта полностью занята terrain | Возврат None → остаётся в резерве |
| Юнит прибыл → сразу стреляет/заряжает | Можно (has_moved = True, но shooting/charging разрешены) |
| Advance после Deep Strike | Нельзя (has_moved = True) |
| Юнит погиб в резерве (не должен случаться) | Просто не размещается |
