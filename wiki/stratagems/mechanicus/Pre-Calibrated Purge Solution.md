# Pre-Calibrated Purge Solution

Предварительно откалиброванный раствор для зачистки! Skitarii получают переброс попаданий по целям в зоне высадки врага. В детачменте [[Rad-Zone Corps]] это поощряет агрессивное развёртывание и подавление вражеских резервов.

**Тактика:** Идеальна для первого хода, когда враг ещё в своей зоне высадки. [[Skitarii Rangers]] с Transuranic Arquebus могут снайперить элитные цели с перебросом попаданий. BATTLELINE отряд делит бонус с соседними Skitarii. Комбинируйте с [[Aggressor Imperative]] для быстрого продвижения в зону врага.
