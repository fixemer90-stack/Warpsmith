# Shield Matrix

Персонаж получает 4+ invulnerable save (неуязвимый спасбросок).

Защитный энхансмент для детачмента [[Retaliation Cadre]]. Обеспечивает базовую защиту от высоко-AP оружия для командира баттлсьюта. Критически важен для [[Commander In Coldstar Battlesuit]], у которого нет встроенного invuln, или [[Commander In Enforcer Battlesuit]] для усиления survivability. Позволяет агрессивно выдвигать командира в зону 9" для активации Deadly Precision.

Ссылки: [[Retaliation Cadre]], [[Commander In Coldstar Battlesuit]], [[Commander In Enforcer Battlesuit]], [[Commander Farsight]], [[Crisis Battlesuits]], [[T'au Empire]]
