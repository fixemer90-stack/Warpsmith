# F2.5: Game Loop — Command -> Movement -> Shooting -> Charge -> Fight

**Файл:** `backend/engine/scenario.py` (создать)
**Зависимости:** F2.1 (Game State), F2.3 (LoS), F2.4 (Missions), F1.6 (Combat)
**Часы:** 6h

**Цель (SMART):** Реализовать полный цикл раунда: 6 фаз (Command -> Movement -> Shooting -> Charge -> Fight -> Morale), каждая вызывает соответствующие методы engine. Критерий: вызов run_round(state) проходит все фазы без ошибок, возвращает обновлённый GameState с применёнными CP, перемещениями, стрельбой, зарядами, боем и VP. 6 часов.

## Core Loop

```python
from dataclasses import dataclass, field

@dataclass
class RoundLog:
    """Лог одного раунда."""
    round: int
    events: list[dict] = field(default_factory=list)
    phase_logs: dict[str, list[dict]] = field(default_factory=dict)


def run_round(state: GameState, map_: GameMap,
              mission: Mission, log: RoundLog) -> GameState:
    """Пройти один полный раунд игры."""

    # Command Phase — оба игрока
    state = command_phase(state, mission, log)

    # Ход P1
    state.turn = 1
    state.phase = "movement"
    state = movement_phase(state, map_, log)
    state.phase = "shooting"
    state = shooting_phase(state, map_, log)
    state.phase = "charge"
    state = charge_phase(state, map_, log)
    state.phase = "fight"
    state = fight_phase(state, map_, log)

    # Ход P2
    state.turn = 2
    state.phase = "movement"
    state = movement_phase(state, map_, log)
    state.phase = "shooting"
    state = shooting_phase(state, map_, log)
    state.phase = "charge"
    state = charge_phase(state, map_, log)
    state.phase = "fight"
    state = fight_phase(state, map_, log)

    # Morale Phase
    state.phase = "morale"
    state = morale_phase(state, log)

    # End of round
    state.round += 1
    state.battle_round += 1
    state.turn = 1

    return state


def run_game(state: GameState, map_: GameMap,
             mission: Mission, max_rounds: int = 5) -> list[RoundLog]:
    """Полная игра: 5 раундов."""
    logs = []
    for r in range(max_rounds):
        log = RoundLog(round=r + 1)
        state = run_round(state, map_, mission, log)
        logs.append(log)
        # Check end conditions
        if _check_game_end(state, mission):
            break
    return logs
```

    """Command Phase: CP generation + Battle-shock + VP scoring."""

## Helper Functions

```python
def _find_valid_targets(unit: UnitState, opponent: RosterState,
                        map_: GameMap) -> list[UnitState]:
    """Найти цели в LoS и range."""
    targets = []
    for target in opponent.units:
        if target.squad_size <= 0:
            continue
        if not has_los(map_, unit.position.x, unit.position.y,
                       target.position.x, target.position.y):
            continue
        # TODO: check weapon range
        targets.append(target)
    return targets

def _check_game_end(state: GameState, mission: Mission) -> bool:
    """Проверить условия окончания игры."""
    if state.battle_round >= mission.config.max_rounds:
        return True
    if state.victory_points[1] >= 100 or state.victory_points[2] >= 100:
        return True
    if _is_army_wiped(state):
        return True
    return False
```
