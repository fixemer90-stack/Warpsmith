# Solid-image Projection Unit

Персонаж получает способность Lone Operative: не может быть целью дальнобойных атак, если стреляющий находится дальше 12".

Энхансмент защиты дальнего боя для детачмента [[Kauyon]]. Позволяет персонажу безопасно выдвигаться для споттинга и использования Kauyon-правил. Отлично сочетается с [[Commander Shadowsun]] (у уже есть Lone Operative, но на другом персонаже даёт защиту). Рекомендуется на [[Commander In Coldstar Battlesuit]] для агрессивного скаутинга или [[Darkstrider]] для поддержки Pathfinders.
