# Counter-okn

Позволяет вашему отряду немедленно ответить огнём после того, как по нему пострелял враг. Это **не** реакция — стратагема используется в вашу фазу стрельбы, когда вражеский отряд закончил стрелять по вашему.

**Важно:** цель ответного огня — только тот отряд, который только что стрелял. Нельзя стрелять в другой отряд. Работает по любым отрядам, включая персонажей, транспорты и монстров. Можно комбинировать с [[Command Re-roll]] для переброса кубиков при ответном огне.
