# F4.1: Faction Browser with Category/PTS Filter

**Файлы:** `web/templates/partials/faction_browser.html` (создать),
`web/static/faction_browser.js` (создать), `web/routes/api.py` (расширение)
**Зависимости:** F2.11 (Team Builder UI), F1.8 (Icon Map / ICON_MAP)
**Часы:** 4h

**Цель (SMART):** Создать полноценный браузер фракций/юнитов с фильтрацией по категории
и PTS-диапазону. Критерий: 6 фильтров (faction, category, PTS min, PTS max, name search,
role — Leader/Transport/Battleline), результаты обновляются через HTMX за <500ms,
пустое состояние отображает "No units match your filters". 4 часа.

## Alpine.js Controller (faction_browser.js)

```javascript
// web/static/faction_browser.js
function factionBrowser() {
    return {
        // Sync state with HTMX filters (for Alpine-reactive elements)
        selectedFaction: '',
        selectedCategory: '',
        ptsMin: null,
        ptsMax: null,
        searchQuery: '',
        selectedRole: '',

        // Quick filter presets
        presets: {
            cheap: { label: 'Under 100pts', pts_max: 100 },
            elite: { label: '100-250pts', pts_min: 100, pts_max: 250 },
            hq: { label: 'Characters', role: 'leader' },
            battleline: { label: 'Battleline', role: 'battleline' },
        },

        applyPreset(preset) {
            Object.assign(this, preset);
            // Trigger HTMX re-request
            htmx.trigger('#faction-browser', 'preset-change');
        },

        get activeFilterCount() {
            let count = 0;
            if (this.selectedFaction) count++;
            if (this.selectedCategory) count++;
            if (this.ptsMin || this.ptsMax) count++;
            if (this.searchQuery) count++;
            if (this.selectedRole) count++;
            return count;
        },

        clearFilters() {
            this.selectedFaction = '';
            this.selectedCategory = '';
            this.ptsMin = null;
            this.ptsMax = null;
            this.searchQuery = '';
            this.selectedRole = '';
            htmx.trigger('#faction-browser', 'filters-cleared');
        },
    };
}
```

## State Handling

| Состояние | Поведение |
|-----------|-----------|
| **Загрузка** | HTMX spinner + skeleton grid (6 серых прямоугольников с пульсацией) |
| **Пусто** | `No units match your filters` — иконка, текст, совет по изменению фильтров |
| **Ошибка сети** | `Failed to load units. Retry?` — кнопка Retry, hx-trigger="click" на `/api/units/browse` |
| **1 результат** | Одна карточка по центру, не растягивается |
| **>100 результатов** | Пагинация (Prev/Next + page X of Y) |

## Тесты

```python
def test_browse_units_no_filters(client):
    """Без фильтров возвращает все юниты."""
    resp = client.get("/api/units/browse")
    assert resp.status_code == 200
    data = resp.json()
    assert "items" in data
    assert data["total"] > 0

def test_browse_units_faction_filter(client):
    resp = client.get("/api/units/browse?faction=orks")
    data = resp.json()
    for item in data["items"]:
        assert item["faction"] == "orks"

def test_browse_units_pts_range(client):
    resp = client.get("/api/units/browse?pts_min=50&pts_max=100")
    data = resp.json()
    for item in data["items"]:
        assert 50 <= item["points"] <= 100

def test_browse_units_empty_state(client):
    resp = client.get("/api/units/browse?pts_min=99999")
    data = resp.json()
    assert data["total"] == 0
    assert len(data["items"]) == 0

def test_browse_units_search(client):
    resp = client.get("/api/units/browse?search=warboss")
    data = resp.json()
    assert all("warboss" in item["name"].lower() for item in data["items"])

def test_browse_units_role_leader(client):
    resp = client.get("/api/units/browse?role=leader")
    data = resp.json()
    assert len(data["items"]) > 0  # есть хотя бы 1 лидер

def test_faction_browser_page(client):
    resp = client.get("/team-builder")
    assert resp.status_code == 200
    assert "faction-browser" in resp.text or "x-data=\"factionBrowser\"" in resp.text
```
