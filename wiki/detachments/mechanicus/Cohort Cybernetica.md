# Cohort Cybernetica

## Detachment Rule: Cyber-Psalm Programming

В начале каждой фазы боя выберите один Protocol для всех дружественных KASTELAN ROBOTS юнитов. Эффект длится до начала вашей следующей фазы:

- **Aegis Protocol** — юнит получает 4+ invulnerable save.
- **Accelerate Protocol** — юнит получает +1" к Move и +1" к Charge.
- **Annihilate Protocol** — юнит получает +1 к Armour Penetration для всех атак.

## Enhancements (4)

- **Cybernetic Controller** — один дружественный KASTELAN ROBOT юнит получает +1 к характеристикам Weapon Skill и Ballistic Skill.
- **Override Command** — один раз за бой, в начале любой фазы вы можете сменить текущий Protocol на другой без ожидания следующей фазы.
- **Blackout-key** — в вашей Command Phase выберите одну вражескую модель в 12" от носителя. Она теряет все auras и abilities до начала вашей следующей Command Phase.
- **Machine-Spirit Node** — дружественные KASTELAN ROBOTS юниты получают Feel No Pain 5+.

## Stratagems (6)

- **Automated Defences** — в вашей Shooting Phase один KASTELAN ROBOTS юнит получает [SUSTAINED HITS 1]. (CP1)
- **Tectonic Stomp** — в вашей Fight Phase один KASTELAN ROBOTS юнит атакует с +1 AP в ближнем бою. (CP1)
- **Override Sentinels** — в начале вашей Command Phase один KASTELAN ROBOTS юнит может игнорировать штрафы к Move за Difficult Terrain до вашей следующей Command Phase. (CP1)
- **Data-ward** — до начала вашей следующей Command Phase один KASTELAN ROBOTS юнит получает -1 к урону от всех атак (минимально 1). (CP1)
- **Reactive Reprogramming** — в любой фазе, когда KASTELAN ROBOTS юнит получает урон от ranged-атаки, вы можете дать ему +1 к Save до конца фазы. (CP1)
- **Crushing Advance** — в вашей Charge Phase один KASTELAN ROBOTS юнит может перебрасывать Charge. (CP1)
